# 0〜2歳 育児チュートリアル完全ガイド

安全・手続き・購入品・発達・教育・遊びの実践ガイド

日本版 / 2026年6月版

> 対象：妊娠中〜子どもが2歳頃まで  
> 目的：出産前に何を準備し、産後に何を見て、何を買い、どう関わるかを迷わない状態にする  
> 方針：公的機関・小児医学・発達心理・運動発達研究・製品公式情報をもとに、家庭で使える形に落とし込む

## 読み方

- 最初から読む場合は [はじめに](docs/00_はじめに.md) から進んでください。
- 必要なところだけ読む場合は、下の目次から章を選んでください。
- 不安が強い時は、まず [第3章　安全対策](docs/03_安全対策.md) と [第12章　小児科探し・予防接種予約・保育園申請](docs/12_小児科探し_予防接種予約_保育園申請.md) を読んでください。
- 生活を具体化したい時は、[第6章　月齢別ロードマップ](docs/06_月齢別ロードマップ.md) と [第10章　1日の流れ](docs/10_1日の流れとシーン別必要品.md) を開いてください。
- 教育・遊びをすぐ実践したい時は [第9章　教育・遊び・おもちゃ 実践編](docs/09_教育_遊び_おもちゃ.md)、根拠から深く読みたい時は [第13章　発達教育 研究編](docs/13_0〜2歳の発達教育_研究サーベイと家庭実践.md) を開いてください。

## この本の進め方

この本は、育児を「気合」ではなく「運用」に落とすためのチュートリアルです。

1. 命に関わる安全を先に固定する
2. 小児科・予防接種・保育園など締切があるものをカレンダー化する
3. 月齢ごとの生活をざっくり把握する
4. 授乳、離乳食、皮膚、睡眠、遊びで迷った時に該当章へ戻る
5. 完璧を目指さず、赤ちゃんと親の状態を見て調整する

## 本の構成

この本は、次の5つの部として読めるようにしています。

| 部 | 章 | 役割 |
|---|---|---|
| まず全体像 | はじめに、第1章 | 育児の優先順位と判断軸を作る |
| 産前に仕組み化 | 第2章、第12章、付録A | 手続き、小児科、予防接種、保育園、購入準備を先回りする |
| 命と生活を守る | 第3章、第7章、第8章、第10章、第11章 | 安全、授乳、離乳食、皮膚、風呂、歯、親の運用を回す |
| 買い物を整理する | 第4章、第5章、付録B、付録C | 本当に必要な物、買い足す時期、後回しでよい物を分ける |
| 発達と教育 | 第6章、第9章、第13章 | 月齢別ロードマップ、遊びの実践、研究サーベイをつなげる |

| テーマ | まず読む章 | 深く見る章 | 使い分け |
|---|---|---|---|
| 安全睡眠 | 第3章 | 第4章、付録C | 第3章でルール、第4章で買い物、付録Cで避ける物を確認 |
| 月齢別の発達 | 第6章 | 第9章、第13章 | 第6章で時期、第9章で遊び、第13章で根拠を確認 |
| 授乳・離乳食 | 第7章 | 第10章 | 第7章で知識、第10章で1日の流れに落とす |
| お風呂・歯・予防接種 | 第8章 | 第12章 | 第8章でケア、第12章で予約と運用を確認 |

| 親の不安 | 第11章 | 第3章、第12章 | 第11章で運用、第3章で事故予防、第12章で相談先を作る |

## 重要な前提

- 本書は一般的な育児・購入・発達支援のガイドです。
- 医療判断、投薬、アレルギー対応、発達診断、緊急対応は、産院・小児科・自治体・救急相談に従ってください。
- 発熱、呼吸が苦しそう、哺乳できない、意識がぼんやり、けいれん、誤飲疑い、転落後の異常、強いアレルギー症状などは、自己判断で様子見せず医療機関へ相談してください。
