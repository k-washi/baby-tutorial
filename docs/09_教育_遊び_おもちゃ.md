# 第9章　教育・遊び・おもちゃ 実践編

## 9-1. 0〜2歳の教育で最も重要なこと

0〜2歳では、座学や暗記より、次の力を育てます。

| 領域 | 育てるもの |
|---|---|
| 愛着 | 親が応答してくれる安心感 |
| 言葉 | 聞く、まねる、指差す、伝える |
| 運動 | 寝返り、腹ばい、ハイハイ、歩く |
| 手指 | 握る、離す、つまむ、入れる |
| 認知 | 原因と結果、予測、分類 |
| 社会性 | まね、順番、共同注意 |
| 自己調整 | 待つ、切り替える、落ち着く |

### 用語補足

| 用語 | 意味 |
|---|---|
| 教育 | ここでは文字や数を教え込むことではなく、脳・身体・言葉・社会性が育つ環境を作ること |
| 発達 | 月齢とともに、できること、感じ方、体の使い方、他者との関わりが変わること |
| 刺激 | 音や光をたくさん与えることではなく、赤ちゃんが処理でき、反応できる経験のこと |
| 応答性 | 赤ちゃんの表情・声・視線・指差し・泣きに、大人が気づいて返すこと |
| 共同注意 | 赤ちゃんと大人が同じものを見て、「これだね」と注意を共有すること |
| 会話ターン | 赤ちゃんが声や表情を出し、大人が返し、また赤ちゃんが返す往復 |
| 実行機能 | 注意を向ける、待つ、切り替える、衝動を少し止めるなどの力 |

研究者の視点で見ると、0〜2歳の教育は「情報をたくさん入力すること」ではありません。  
赤ちゃんが見た、触った、動いた、声を出した。それに大人が意味をつけて返し、赤ちゃんがもう一度試す。この往復が学習単位です。

### 研究の読み方

育児情報では「この教材で賢くなる」「動画で英語脳」など、強い言い方が出てきます。  
研究を見る時は、次のように分けます。

| 種類 | 何がわかるか | 注意点 |
|---|---|---|
| 観察研究 | ある行動と発達が関連するか | 因果とは限らない。家庭環境などの交絡が残る |
| 介入研究 | ある関わりを増やすと変化が出るか | 対象年齢・家庭背景・期間に注意 |
| ガイドライン | 専門家が複数研究を踏まえて推奨すること | 国や時期で更新される |
| 商品広告 | 商品の価値を伝える情報 | 発達効果の根拠が弱いことが多い |

本書では、家庭での優先順位を次のように置きます。

1. 安全な睡眠、栄養、皮膚、予防接種
2. 親が倒れない運用
3. 応答的な関わり
4. 会話、読み聞かせ、歌
5. 床で自由に動く時間
6. 手指を使う遊び、ごっこ遊び
7. 画面や教材は最後に検討

この章は、家庭で今日から使うための実践編です。  
研究サーベイ、エビデンス強度、月齢別の実践メニューまで深く読む場合は、[第13章　発達教育 研究編：サーベイと家庭実践](13_0〜2歳の発達教育_研究サーベイと家庭実践.md)に進んでください。

## 9-2. Serve and Return：赤ちゃんの反応に返す

Serve and Returnは、直訳すると「サーブして、返す」です。  
テニスのラリーのように、赤ちゃんが出した小さなサインに、大人が返す関わりを指します。日本語では「応答的なやりとり」と考えるとわかりやすいです。

### やり方

1. 赤ちゃんが見る・声を出す・指差す
2. 親が気づく
3. 言葉・表情・動作で返す
4. 赤ちゃんの次の反応を待つ
5. もう一度返す

### 例

| 赤ちゃん | 親の返し |
|---|---|
| 「あー」 | 「あーって言ったね」 |
| 犬を指差す | 「犬だね。白い犬だね」 |
| ボールを見る | 「ボール見つけたね。転がす？」 |
| 泣く | 「眠かったね。抱っこするね」 |

### 根拠

ハーバード大学は、こうした応答的なやりとりが脳の発達に重要だと説明しています<sup>[1](#ref-harvard-serve)</sup>。また、親子の会話ターンを増やす介入によって乳児の言語発達が促進された研究もあります<sup>[2](#ref-pnas-lang)</sup>。

### なぜ効くのか

赤ちゃんは、まだ抽象的な説明を理解できません。  
しかし、「自分が声を出すと親が返す」「見たものに名前がつく」「泣くと抱っこされる」「指差すと大人も見る」という因果関係は学べます。

この往復には、少なくとも4つの意味があります。

| 視点 | 意味 |
|---|---|
| 言語 | 見ているものと言葉が結びつく |
| 情緒 | 自分のサインが届くという安心感が育つ |
| 注意 | 同じ対象へ注意を向ける練習になる |
| 自己調整 | 泣く、待つ、落ち着く経験が積み上がる |

重要なのは、大人が一方的に話し続けることではありません。  
赤ちゃんの反応を待つ「間」が必要です。声が出ない月齢でも、視線、手足の動き、表情、口の形が返事になります。

## 9-3. 親がよく話す

### ただし、聞かせっぱなしではない

テレビ、動画、音声教材を流すだけでは、親子の会話の代わりになりません。  
重要なのは、赤ちゃんに向けて話し、赤ちゃんの反応を待つことです。

研究で重視されるのは、単語数だけではなく、会話ターンです。  
会話ターンとは、赤ちゃんの声・表情・動きに大人が返し、その返しに赤ちゃんがまた反応する往復です。成人と子どもの会話ターンは、言語関連脳ネットワークの発達とも関連すると報告されています<sup>[3](#ref-jns-lang)</sup>。

### 話しかけの質

| 低い効果になりやすい | 効果が出やすい |
|---|---|
| 大人同士の会話を聞かせるだけ | 赤ちゃんに向けて話す |
| 動画や音声を流すだけ | 赤ちゃんの反応に返す |
| 「これは何？」を連発する | 「赤い車だね。走ったね」と実況する |
| 正解を求める | 気づき、感情、動作に言葉をつける |
| 早口で詰め込む | 間を置いて反応を待つ |

0〜2歳では、「教える」より「言葉を添える」方が自然です。  
おむつ替え、授乳、入浴、散歩、食事、着替えはすべて教材になります。

### 毎日の実況中継

| 場面 | 言葉 |
|---|---|
| おむつ替え | 「おむつ替えるよ。きれいになったね」 |
| 授乳 | 「お腹すいてたね。飲めたね」 |
| 散歩 | 「車だね。赤いね。速いね」 |
| 入浴 | 「あったかいね。泡だよ」 |
| 食事 | 「にんじんだよ。やわらかいね」 |
| 着替え | 「右手を通すよ。できたね」 |

### 研究者視点の実践

1日中話し続ける必要はありません。  
家庭で見る指標は、「今日、赤ちゃんが発したサインに何回返せたか」です。

- [ ] 赤ちゃんが見たものに名前をつけた
- [ ] 声を出したら、まねして返した
- [ ] 指差しや視線の先を一緒に見た
- [ ] 泣いた理由を推測して言葉にした
- [ ] 絵本や散歩で、同じものを見ながら話した

## 9-4. 絵本は0か月から

AAPは、出生直後から親子で読み聞かせを始めることを推奨しています<sup>[4](#ref-aap-reading)</sup>。  
0歳では内容理解より、親の声、リズム、ページをめくる動作、同じものを見る経験が重要です。

ここでいう絵本は、文字学習の教材ではありません。  
赤ちゃんにとっては、親の声、絵、リズム、指差し、ページをめくる動きがまとまった「共同注意の道具」です。

### 年齢別の絵本

| 月齢 | おすすめ |
|---|---|
| 0〜3か月 | コントラストが強い、リズムがある |
| 3〜6か月 | 布絵本、音、触感 |
| 6〜12か月 | 厚紙絵本、擬音、繰り返し |
| 1〜2歳 | 動物、乗り物、生活習慣、ごっこ遊び |

### チェックリスト

- [ ] 1日1冊でも読む
- [ ] 同じ本を何度読んでもよい
- [ ] 指差しに反応する
- [ ] 読み聞かせ中に脱線してよい
- [ ] 破ってもよい厚紙絵本を用意する
- [ ] 寝る前ルーティンに入れる

### 読み方のコツ

| 子どもの反応 | 親の返し |
|---|---|
| ページをすぐめくる | 「次に行きたいんだね」と合わせる |
| 同じページを触る | 「ここが気になるね。犬だね」 |
| 途中で離れる | 終わりでよい。短くても成立 |
| 同じ本ばかり選ぶ | 予測できることが安心と学習になる |
| 絵を指差す | 名前、色、動き、感情を言葉にする |

絵本の効果は、冊数の多さだけでは決まりません。  
同じ本を何度も読み、同じ言葉を聞き、同じ場面を親と共有することで、予測と言葉の結びつきが強くなります。

## 9-5. スクリーン時間

スクリーン時間とは、テレビ、スマートフォン、タブレット、パソコン、動画、アプリ、ゲームなど、画面を見て過ごす時間のことです。  
育児ガイドラインで問題にされるのは、特に、座ったまま画面を見る「座位スクリーン時間」です。

### 用語補足

| 用語 | 意味 |
|---|---|
| スクリーン時間 | テレビ、スマホ、タブレット、動画、アプリなど画面を見る時間 |
| 座位スクリーン時間 | 座ったまま、体をあまり動かさず画面を見る時間 |
| 受動視聴 | 子どもが画面をただ見ている状態 |
| 共同視聴 | 親が一緒に見て、内容を説明したり会話したりする見方 |
| バックグラウンドテレビ | 子どもが直接見ていなくても、部屋でテレビや動画が流れている状態 |
| ビデオ通話 | 離れた家族などとリアルタイムにやりとりする通話 |

WHOは、1歳未満のスクリーン時間を推奨していません。2歳でも座りっぱなしのスクリーン時間は少ないほどよいとしています<sup>[5](#ref-who-activity)</sup>。AAPも18か月未満はビデオ通話を除いてメディア利用を避けることを勧めています<sup>[6](#ref-aap-media)</sup>。

### なぜ注意するのか

スクリーンの問題は、「画面を見ると脳が壊れる」という単純な話ではありません。  
主な問題は、置き換えです。画面を見ている時間は、床で動く、手で触る、親と会話する、絵本を見る、眠る、食事に集中する時間を奪いやすくなります。

| 視点 | 起きやすい問題 |
|---|---|
| 言語 | 一方向の音声になり、会話ターンが減る |
| 注意 | 速い切り替えや強い刺激に慣れやすい |
| 睡眠 | 寝る前の刺激で寝つきが悪くなる |
| 食事 | 満腹感や一口量に注意が向きにくい |
| 運動 | 床で動く時間が減る |
| 親子関係 | 泣きへの応答が動画に置き換わる |

JAMA Pediatricsのカナダの縦断研究では、2〜5歳の子どもで、スクリーン時間と発達スクリーニング成績の関連が検討されています<sup>[8](#ref-jama-screen-2019)</sup>。日本の東北メディカル・メガバンク計画のコホート研究でも、1歳時点のスクリーン時間が長いほど、2歳・4歳時点のコミュニケーションや問題解決などの発達遅れと関連する用量反応関係が報告されています<sup>[9](#ref-jama-screen-japan-2023)</sup>。

ただし、これらは主に観察研究です。  
スクリーン時間そのものだけでなく、親の疲労、家庭のストレス、睡眠、屋外活動、親子の会話量などが絡みます。研究者として読むなら、「スクリーンは絶対悪」と短絡せず、「スクリーンが何を置き換えているか」「親子のやりとりを減らしていないか」を見るのが実践的です。

### 研究データをどう読むか

数値は「この時間を超えたら必ず発達が遅れる」という意味ではありません。  
集団として見た時に、スクリーン時間が長い群で発達スクリーニング上のリスクが高い、という読み方をします。

| 研究 | 対象 | 主な結果 | 家庭での読み方 |
|---|---|---|---|
| Madiganら 2019 | カナダの母子2441組。24・36・60か月を追跡 | 24か月・36か月時点のスクリーン時間が長いほど、後の発達スクリーニング得点が低い方向に関連 | 2歳前後から「増えすぎ」を防ぐ。発達が気になる時ほど動画で静かにさせ続けない |
| Takahashiら 2023 | 日本の母子7097組。1歳時点のスクリーン時間と2歳・4歳の発達を検討 | 1歳で4時間以上の群は、1時間未満の群と比べて2歳のコミュニケーション遅れが多く、調整後オッズ比は4.78。4歳時点でもコミュニケーション遅れの調整後オッズ比は2.68 | 1歳台の長時間視聴は避ける。特に言葉・指差し・共同注意が気になる家庭は、画面より人との往復を増やす |

ここで出てくる「オッズ比」は、ある出来事が起きる起こりやすさの比です。  
たとえばオッズ比4.78は、「発達遅れの確率が4.78倍」と単純に読むものではありません。もともとの割合、調整した要因、測定方法によって解釈が変わります。それでも、1歳時点で4時間以上という長時間のスクリーン時間は、家庭で減らす優先度が高いサインです。

研究から実践に落とす時の結論は、次の3つです。

1. 0〜1歳台は、画面で学ばせるより、睡眠・食事・床遊び・会話ターンを守る。
2. 使う時は、子どもだけで見せず、大人が一緒に意味づけする。
3. 親が限界の日の短時間利用は責めないが、毎日の主要な育児手段にしない。

### 現実的なルール

| 年齢 | 方針 |
|---|---|
| 0〜18か月 | 原則なし。ビデオ通話は可 |
| 18〜24か月 | 使うなら短時間、親と一緒 |
| 2歳以降 | 1時間以内を目安。内容を選ぶ |

### 使うなら守る条件

- [ ] 子どもだけに見せっぱなしにしない
- [ ] 親が内容を知っているものにする
- [ ] 速すぎる編集、強すぎる音、広告が多いものを避ける
- [ ] 食事中、寝る前、癇癪のたびの常用は避ける
- [ ] 見た後に、実物、絵本、会話、体を動かす遊びへつなげる
- [ ] バックグラウンドでテレビをつけっぱなしにしない

ビデオ通話は例外扱いされることがあります。  
理由は、画面越しでも相手が子どもの声や表情に反応し、会話ターンが成立しやすいからです。ただし、これも長時間の受動視聴とは別物として考えます。

### やらない方がよいこと

- [ ] 食事中に動画
- [ ] 寝る前の動画
- [ ] 泣き止ませ目的で常用
- [ ] 親子の会話の代わりにする
- [ ] 「英語教育」として長時間流す

### 英語動画・知育動画について

0〜2歳では、英語動画を長く流すより、親が日本語でもよいので応答的に話す方が優先です。  
言語は、音を聞くだけでなく、視線、口の動き、場面、感情、順番、共同注意と結びついて学びます。動画を使うなら、親が一緒に見て「犬だね」「jumpしてるね」と現実の遊びや絵本につなげます。

## 9-6. おもちゃの選び方

AAPは、幼児期のおもちゃ選びでは、親子の相互作用、想像力、問題解決、身体活動を促すものを重視し、電子的・デジタル玩具が必ずしも発達に優れるわけではないと説明しています<sup>[7](#ref-aap-toys)</sup>。

ハーバード大学Center on the Developing Childも、乳幼児期の遊びを、応答的な関係、実行機能、自己制御、レジリエンスの土台に関わる活動として位置づけています<sup>[10](#ref-harvard-play)</sup>。乳幼児の遊びは、暇つぶしではなく、仮説検証です。落とす、入れる、出す、積む、崩す、転がす、まねる、隠す、探す。これらはすべて、原因と結果を学ぶ実験です。

### 良いおもちゃの条件

- [ ] 親子の会話が増える
- [ ] 子どもが自分で操作できる
- [ ] 遊び方が1つに固定されない
- [ ] 手指を使う
- [ ] 体を動かす
- [ ] まね・ごっこ遊びができる
- [ ] 小パーツがない
- [ ] 洗える・清潔にできる

### ぬいぐるみ・メリーの位置づけ

ぬいぐるみは、0か月から「親が見せて話しかけるおもちゃ」として使えます。  
ただし、1歳までは寝床に入れません。寝ている時の安全ルールは[第3章](03_安全対策.md)を優先します。

メリーは、見る、追う、音を聞く遊びとして使えます。  
ただし、赤ちゃんが手を伸ばして触れる時期になったら、ベッド取り付け型は外します。床置きのプレイジムやメリーは、起きていて大人が見ている時間の遊び場で使います。

| 品目 | 教育的な使い方 | 安全上の注意 |
|---|---|---|
| ぬいぐるみ | 名前をつける、声を当てる、触感を言葉にする | 寝床に入れない。小部品・長い毛・ひもを避ける |
| メリー | 視線を向ける、追視、音への反応 | 手が届かない位置。寝返り・手伸ばしが始まったら外す |
| 床置きプレイジム | 手を伸ばす、蹴る、腹ばい前の遊び | 長時間固定せず、眠ったら寝床へ移す |

### 研究者視点で見るおもちゃ

良いおもちゃは、子どもが能動的に変化を作れます。  
ボタンを押すと一方的に音が鳴るだけの玩具より、積み木、コップがさね、ボール、布、箱のように、子ども自身が試行錯誤できるものの方が、親子の会話も増えやすくなります。

| 玩具の性質 | 学び |
|---|---|
| 入れる・出す | 物の永続性、分類、手指 |
| 積む・崩す | 因果関係、空間認知、失敗への耐性 |
| 転がす | 予測、追視、移動 |
| まねる | 社会性、言葉、ごっこ遊び |
| 隠す・探す | 記憶、期待、共同注意 |
| 描く | 手指、痕跡、表現 |

### 月齢別おすすめ

#### 0〜3か月

- [ ] プーメリー
- [ ] 小さく洗えるぬいぐるみ
- [ ] オーボール
- [ ] 布絵本
- [ ] ラトル
- [ ] 親の顔と声

目的：見る、聞く、握る準備、安心。

#### 3〜6か月

- [ ] ベビージム
- [ ] 鏡
- [ ] 歯固め
- [ ] 布ボール
- [ ] シャカシャカ布

目的：手を伸ばす、寝返り、腹ばい。

#### 6〜9か月

- [ ] コップがさね
- [ ] ボール
- [ ] 厚紙絵本
- [ ] 音の出る布おもちゃ
- [ ] 入れる/出す遊び

目的：座る、ずりばい、因果関係。

#### 9〜12か月

- [ ] くるくるチャイム
- [ ] 積み木
- [ ] 型はめ

## 9-7. ハイハイを促す家庭環境

### やること

- [ ] 床に広いスペースを作る
- [ ] 滑りにくいマットにする
- [ ] 赤ちゃんの少し先におもちゃを置く
- [ ] バウンサー・椅子固定を長時間にしない
- [ ] 親も床に座る
- [ ] 無理に立たせない
- [ ] 成功したら言葉で返す

### やらないこと

- [ ] 早く歩かせるために手を引きすぎる
- [ ] 歩行器に長時間入れる
- [ ] 常に椅子やバウンサーに置く
- [ ] ハイハイを飛ばしたと決めつけて焦る
- [ ] 他の子と比べる

## 9-8. 家庭でできる「発達を伸ばす環境」の作り方

研究者のように家庭を観察するなら、見るべき変数は教材名ではありません。  
次の4つです。

| 変数 | 観察すること |
|---|---|
| 応答 | 子どものサインに大人が返しているか |
| 反復 | 同じ遊びや本をくり返せているか |
| 自由度 | 子どもが自分で操作できるか |
| 安全 | 動ける環境が危険物なしで作れているか |

### 1日の中で入れたい経験

- [ ] 親の顔を見る
- [ ] 声を出したら返してもらう
- [ ] 同じものを一緒に見る
- [ ] 絵本を読む
- [ ] 床で動く
- [ ] 手で触る、入れる、出す、落とす
- [ ] 外の光、風、音を感じる
- [ ] 眠る前に静かなルーティンがある

### 逆に減らしたいもの

- [ ] 画面のつけっぱなし
- [ ] 椅子、バウンサー、ベビーカーへの長時間固定
- [ ] 大人が正解を急かす遊び
- [ ] 月齢より早い知育課題
- [ ] 親が疲れ切るほどの教育ノルマ

0〜2歳の発達を伸ばす家庭は、教材が多い家庭ではありません。  
安全な床、よく返してくれる大人、くり返し読める絵本、手で試せる物、よく眠れる生活リズムがある家庭です。
