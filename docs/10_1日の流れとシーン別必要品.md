# 第10章　1日の流れとシーン別必要品

## 10-0. この章の使い方

赤ちゃんの1日は、理想の時刻表どおりには進みません。  
特に0歳台は、授乳、睡眠、おむつ、泣き、入浴、散歩が日によってずれます。

この章では、1日を「時刻」ではなく「状態のループ」として見ます。

1. 起きる
2. 授乳・食事
3. おむつ
4. 少し遊ぶ、話す、外気に触れる
5. 眠くなる
6. 寝る
7. 起きたらまた観察する

親が見るべき指標は、「予定どおりか」ではなく、次の4つです。

| 指標 | 見ること |
|---|---|
| 入る | 母乳・ミルク・離乳食・水分が取れているか |
| 出る | おしっこ、うんちが出ているか |
| 眠る | 月齢なりに眠れているか、眠すぎて崩れていないか |
| 反応 | 起きている時に、目が合う、声が出る、動く、遊べるか |

迷った時は、詳しい月齢別の発達は[第6章](06_月齢別ロードマップ.md)、授乳・離乳食は[第7章](07_授乳_ミルク_離乳食_アレルギー.md)、お風呂・歯・予防接種は[第8章](08_皮膚_お風呂_歯_予防接種.md)、小児科と保育園は[第12章](12_小児科探し_予防接種予約_保育園申請.md)に戻ります。

## 10-1. 新生児期の1日

| 場面 | 行動 | 必要品 |
|---|---|---|
| 起床 | 授乳・おむつ | おむつ、おしりふき、授乳用品 |
| 午前 | 寝る | 安全な寝床、スリーパー |
| 昼 | 沐浴 | ベビーバス、泡ソープ、バスタオル |
| 午後 | 抱っこ・寝る | 抱っこ紐、授乳クッション |
| 夕方 | 泣きやすい | バウンサー、見守り |
| 夜 | 夜間授乳 | 授乳ライト、哺乳瓶、ミルク |
| 深夜 | 交代で対応 | 共有記録、飲み物、軽食 |

### 新生児期のシーン別チェック

- [ ] おむつ替えセットを寝室とリビングに置く
- [ ] 夜間授乳セットを1か所にまとめる
- [ ] 体温計と母子手帳の場所を固定
- [ ] 親の水分・軽食も用意
- [ ] 赤ちゃんが寝たら親も寝る

### 新生児期の考え方

新生児期は「生活リズムを作る時期」ではなく、「安全に飲ませて、寝かせて、親が倒れないようにする時期」です。  
昼夜の区別はまだ弱く、夜に何度も起きるのは普通です。

この時期の親のタスクは、細かい教育ではありません。

- [ ] 安全な寝床で仰向けに寝かせる
- [ ] 授乳・ミルクの回数と尿を観察する
- [ ] 黄疸、発熱、哺乳不良、ぐったりに気づく
- [ ] 親が交代で眠れる形を作る
- [ ] 来客・家事・SNSを減らす

### 1回の起床でやる順番

| 順番 | やること | 補足 |
|---|---|---|
| 1 | 泣き方と体の様子を見る | 顔色、呼吸、熱っぽさを確認 |
| 2 | おむつを見る | 尿・便は哺乳の重要な手がかり |
| 3 | 授乳・ミルク | 飲みが悪い、むせる、ぐったりは相談 |
| 4 | げっぷ・縦抱き | 必ず出るとは限らない。苦しそうかを見る |
| 5 | 寝床に戻す | 抱っこで寝ても、寝る場所は安全な寝床 |

### 夜間セット

- [ ] おむつ
- [ ] おしりふき
- [ ] 防水シートまたは替えシーツ
- [ ] 授乳クッション
- [ ] 哺乳瓶・液体ミルク・粉ミルク・調乳用品
- [ ] 低照度のライト
- [ ] 親の水
- [ ] 記録アプリまたはメモ

夜間は判断力が落ちます。  
「どこに何があるか」を昼のうちに固定しておくと、夜の事故とストレスが減ります。

## 10-2. 3〜6か月の1日

| 場面 | 行動 | 必要品 |
|---|---|---|
| 朝 | 起床・授乳 | おむつ、服 |
| 午前 | 床遊び | プレイマット、オーボール |
| 午前後半 | 散歩 | ベビーカー、抱っこ紐 |
| 昼 | 昼寝 | 安全な寝床 |
| 夕方 | タミータイム | ベビージム、鏡 |
| 夜 | 入浴・保湿 | ソープ、ローション |
| 寝る前 | 絵本 | 厚紙絵本、スリーパー |

### 3〜6か月の考え方

少しずつ「起きて遊ぶ時間」が伸びます。  
ただし、長く起こして発達を促す時期ではありません。眠いサインが出たら、早めに寝かしつけます。

| 観察するサイン | 意味 |
|---|---|
| あくび、目をこする、視線が外れる | 眠い |
| 反り返る、泣きが強くなる | 疲れすぎ、刺激が多すぎる可能性 |
| 手を見つめる、口に持っていく | 手の発達、探索 |
| 声を出す、笑う | やりとりの準備 |
| うつ伏せで頭を上げる | 首・背中・肩の筋力発達 |

### 1日の中に入れたいこと

- [ ] 起きている時に短いタミータイムを入れる
- [ ] 親の顔を見せて話す
- [ ] 手で触れるおもちゃを置く
- [ ] 午前または夕方に短く散歩する
- [ ] 入浴後に保湿する
- [ ] 寝る前は光・音・画面を減らす

「タミータイム」は、起きていて大人が見ている時に、短時間うつ伏せで遊ぶことです。寝る時の姿勢ではありません。

## 10-3. 7〜12か月の1日

| 場面 | 行動 | 必要品 |
|---|---|---|
| 朝 | 離乳食 | ベビーチェア、エプロン |
| 午前 | ハイハイ遊び | ベビーゲート、プレイマット |
| 昼 | 昼寝 | 安全な寝床 |
| 午後 | 散歩 | ベビーカー、帽子 |
| 夕方 | 手づかみ食べ | 食器、掃除しやすい床 |
| 夜 | 絵本・入眠 | 絵本、スリーパー |

### 7〜12か月の考え方

この時期は、離乳食、移動、誤飲対策が一気に重要になります。  
「食べる練習」と「動く練習」が増える一方で、事故リスクも上がります。

| シーン | 目的 | 注意 |
|---|---|---|
| 離乳食 | 食材、形状、口の動きを学ぶ | 窒息しやすい形を避け、座って見守る |
| 床遊び | ずりばい、ハイハイ、座位、つかまり立ち | 床の小物、コード、家具角を確認 |
| 散歩 | 光、風、音、人を経験する | 暑さ、寒さ、日差し、虫対策 |
| 入浴 | 清潔、皮膚観察、生活リズム | 浴槽で手を離さない |
| 寝る前 | 切り替え、安心 | 画面を使わず、同じ流れを繰り返す |

### 離乳食の日の運用

- [ ] 食前に眠すぎない時間を選ぶ
- [ ] 椅子に深く座らせる
- [ ] 初めての食材は平日午前から昼に少量
- [ ] 食後の皮膚、咳、嘔吐、機嫌を観察
- [ ] 食事中は動画を見せない
- [ ] 食後に床・椅子・手口を洗える導線にする

離乳食は「栄養を全部食べさせる時間」ではなく、母乳・ミルクから幼児食へ移る練習です。  
食べない日があっても、尿、体重、機嫌、月齢相当の発達を合わせて見ます。

## 10-4. 1〜2歳の1日

| 場面 | 行動 | 必要品 |
|---|---|---|
| 朝 | 朝食・着替え | 食器、服収納 |
| 午前 | 外遊び | 靴、帽子、ボール |
| 昼 | 昼食・昼寝 | チェア、寝具 |
| 午後 | ごっこ遊び | 積み木、ままごと |
| 夕方 | 入浴 | 歯ブラシ、保湿剤 |
| 夜 | 絵本・寝る | 絵本、パジャマ、スリーパー |

### 1〜2歳の考え方

1〜2歳は、歩く、触る、まねる、主張する時期です。  
親の言うことを常に理解して従える年齢ではありません。安全な環境を作り、同じルールを短く繰り返します。

### 生活の軸

| 軸 | 家庭でやること |
|---|---|
| 食事 | 手づかみ、スプーン練習、薄味、座って食べる |
| 睡眠 | 昼寝と夜の入眠ルーティンを固定する |
| 運動 | 毎日、歩く・登る・押す・投げる経験を安全に入れる |
| 言葉 | 実物を見ながら短い言葉を返す |
| 歯 | 仕上げ磨きとフッ化物配合歯磨剤を続ける |
| 安全 | 誤飲、転落、やけど、水回りを毎日見る |

### イヤイヤが出た時の基本

「イヤ」は、親を困らせるためだけではなく、自分で選びたい気持ちの表れです。  
選択肢を2つにすると、親の許容範囲内で本人の主体性を残せます。

| 場面 | 言い方 |
|---|---|
| 着替え | 「赤い服と青い服、どっちにする？」 |
| 入浴 | 「歩いて行く？抱っこで行く？」 |
| 歯磨き | 「座って磨く？抱っこで磨く？」 |
| 片付け | 「車から入れる？積み木から入れる？」 |

危険なことは選択式にしません。  
道路、熱いもの、浴室、誤飲物、ベランダでは、説明より先に物理的に止めます。

## 10-5. 1日の記録テンプレート

毎日すべてを細かく記録する必要はありません。  
ただし、授乳が不安、体重が不安、便秘、発熱、離乳食、予防接種後などは、短期間だけ記録すると相談しやすくなります。

| 項目 | 記録例 |
|---|---|
| 授乳・ミルク | 時刻、量、飲み方 |
| 離乳食 | 食材、量、反応 |
| おしっこ | 回数、色、少なさ |
| うんち | 回数、硬さ、血、色 |
| 睡眠 | 寝た時刻、起きた時刻、夜間覚醒 |
| 体調 | 体温、咳、鼻水、発疹、嘔吐 |
| 機嫌 | いつもと違う泣き、ぐったり |
| 薬・保湿 | 使った時刻、量、反応 |

### 小児科に見せやすいメモ

- いつから
- 何回
- どのくらい
- 飲めているか
- 尿が出ているか
- 眠れているか
- 写真や動画があるか

「なんとなく変」でも、時系列にすると医師が判断しやすくなります。

## 10-6. 平日・休日・体調不良日の切り替え

### 平日

- [ ] 朝の準備を最小手数にする
- [ ] おむつ、着替え、保育園用品を前夜に置く
- [ ] 朝食・離乳食は安全に食べられるものを固定化する
- [ ] 帰宅後は入浴、食事、睡眠を優先する

### 休日

- [ ] 生活リズムを崩しすぎない
- [ ] 外遊びや散歩を入れる
- [ ] 新しい食材は受診しやすい日中に試す
- [ ] 親のどちらかが休む時間を予定に入れる

### 体調不良日

- [ ] 食事量より水分と尿を見る
- [ ] 眠れる環境を作る
- [ ] 入浴は無理にしない
- [ ] 体温、症状、尿、飲水量を記録する
- [ ] 迷ったら小児科、夜間休日は#8000等の相談窓口を使う

体調不良の日は、発達、教育、片付け、家事をいったん下げます。  
優先順位は「呼吸、水分、尿、意識、睡眠」です。
