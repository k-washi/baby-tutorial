# 第8章　皮膚・お風呂・歯・予防接種

## 8-1. お風呂

赤ちゃんのお風呂は、「毎日きれいにする儀式」ではなく、皮膚を清潔にし、汗・うんち・ミルク・よだれを落とし、皮膚トラブルを早く見つける時間です。  
大人の感覚で長く温まる必要はありません。赤ちゃんは体が小さく、暑さ寒さの影響を受けやすいため、短く、安全に、洗いすぎないことを優先します。

### 用語補足

| 用語 | 意味 |
|---|---|
| 沐浴 | 主に新生児期に、ベビーバスなどで赤ちゃんだけを洗うこと |
| 入浴 | 大人と同じ浴室や浴槽を使って体を洗うこと |
| 湯温 | お湯の温度。熱すぎると乾燥やのぼせにつながる |
| 保湿 | 入浴後などにローションやクリームで皮膚の水分を逃げにくくすること |
| 押し拭き | タオルでこすらず、押さえるように水分を取ること |

### 基本

- ぬるめのお湯
- 泡でやさしく洗う
- こすらない
- すすぎ残しを避ける
- 押し拭き
- 早めに保湿

### 月齢別：お風呂・沐浴の考え方

| 時期 | 基本方針 | 親が見るポイント |
|---|---|---|
| 0〜1か月 | ベビーバスなどで短時間の沐浴が中心 | へその状態、体温、顔色、親の手順 |
| 1〜3か月 | 生活に合わせて沐浴から入浴へ移行 | 首を支える、洗う人と受け取る人の分担 |
| 4〜6か月 | よだれ・首のしわ・手の中を丁寧に洗う | 寝返り前後で浴室内の転倒・落下に注意 |
| 7〜12か月 | 動きが増えるため、浴室の安全対策を強化 | つかまり立ち、浴槽への転落、石けん誤飲 |
| 1〜2歳 | 自分で洗いたがるが、仕上げは大人 | 浴槽内で立つ、滑る、湯を飲む、目を離さない |

### 0〜1か月：沐浴は「短く、安定して、冷やさない」

新生児期は、赤ちゃんも親もお風呂に慣れていません。  
目的は、長く温めることではなく、汗、うんち、ミルク汚れを落とし、皮膚の赤みやじゅくじゅくを見つけることです。

- [ ] 室温を寒すぎない状態にする
- [ ] 着替え、おむつ、保湿剤、タオルを先に広げておく
- [ ] 顔、首、わき、手のひら、足の付け根、おしりを確認する
- [ ] へその周りはこすらず、産院の指示に従う
- [ ] 親が怖い日は無理せず、体を拭く日があってもよい

赤ちゃんは体温調節が未熟です。  
「きれいに洗う」ことに集中しすぎて裸の時間が長くなるより、短く洗って、押し拭きし、早めに服を着せる方が安定します。

### 1〜3か月：沐浴から入浴へ移行する時期

1か月健診で問題がなければ、大人と同じ浴室での入浴へ移行する家庭が増えます。  
ただし、首がまだ不安定な時期なので、洗う人、受け取る人、保湿する人の流れを決めておくと安全です。

- [ ] 首と後頭部を支える
- [ ] 浴室の床に赤ちゃんを直置きしない
- [ ] 親が先に体を洗う場合、赤ちゃんの待機場所を決める
- [ ] 湯船は短時間にする
- [ ] 入浴後はすぐ保湿する

この時期は「毎日湯船に入れなければならない」と考えなくてよいです。  
汗、うんち、吐き戻し、よだれ汚れを清潔にし、親が安全に続けられる方法を優先します。

### 4〜6か月：しわと手の中が汚れやすい

よだれが増え、手を口に入れることが増えます。  
首のしわ、手のひら、指の間、耳の後ろ、わき、足の付け根に汚れが残りやすくなります。

- [ ] 首のしわを開いて泡で洗う
- [ ] 手のひらのほこりやミルク汚れを洗う
- [ ] よだれかぶれは洗って保湿する
- [ ] 寝返りが始まったら、脱衣所や浴室で目を離さない

赤くなっている場所を強くこすると悪化します。  
泡で浮かせて、よくすすぎ、押し拭きしてから保湿します。

### 7〜12か月：浴室は事故リスクが上がる

おすわり、ずりばい、つかまり立ちが始まると、浴室は一気に危険になります。  
水深が浅くても溺れるリスクがあり、浴槽、洗面器、残り湯には注意します。

- [ ] 浴槽に残り湯をためたままにしない
- [ ] 浴室のドアを閉め、子どもだけで入れないようにする
- [ ] 石けん、シャンプー、カミソリ、掃除用品を手の届かない場所へ移す
- [ ] つかまり立ち中は滑り止めや親の手で支える
- [ ] おもちゃを使う場合も、目を離さない

この時期のお風呂は、清潔ケアと同時に水の事故対策です。  
「一瞬だから大丈夫」と思う場面ほど、先にタオルや着替えを準備して、途中で取りに行かない運用にします。

### 1〜2歳：自分でやりたい気持ちを使う

1歳を過ぎると、洗う、流す、タオルで拭くなどを自分でやりたがる子が増えます。  
任せる部分を作るのはよいですが、洗い残し、すすぎ残し、転倒、湯を飲む行動は大人が見ます。

- [ ] 自分で洗う部分を決める
- [ ] 顔や頭を嫌がる時は、短く予告してから洗う
- [ ] 仕上げ洗いは大人が行う
- [ ] 浴槽で立たない、走らないを毎回同じ言葉で伝える
- [ ] 入浴後の保湿を習慣にする

嫌がる日は、全身を完璧に洗い切るより、汚れやすい場所を優先します。  
おしり、首、手、顔、汗をかいた部分が清潔なら、翌日に立て直せます。

### 毎日沐浴・入浴が必要か

生活環境・季節・汗・汚れによります。  
洗いすぎると乾燥する子もいるため、肌状態を見て調整します。汗やうんち汚れ、よだれ汚れは清潔にします。

判断は次のようにします。

| 状態 | 対応 |
|---|---|
| 汗をかいた、うんち汚れ、吐き戻し、よだれが多い | 洗う、または部分洗い |
| 乾燥が強い、湿疹が悪化している | 洗い方・保湿・受診を見直す |
| 親が限界、赤ちゃんが眠すぎる | 体拭きや部分洗いでもよい |
| 発熱・体調不良 | 入浴可否を小児科や自治体相談に確認 |

## 8-2. 保湿と湿疹

赤ちゃんの皮膚は、大人より薄く、乾燥や刺激の影響を受けやすいです。  
保湿は「美容」ではなく、皮膚のバリアを守る日常ケアです。皮膚のバリアが崩れると、かゆみ、掻き壊し、よだれかぶれ、おむつかぶれが起きやすくなります。

### 観察ポイント

- [ ] 赤み
- [ ] じゅくじゅく
- [ ] かゆそう
- [ ] 掻き壊し
- [ ] よだれかぶれ
- [ ] おむつかぶれ
- [ ] 乾燥で粉をふく

### 月齢別：保湿で見る場所

| 時期 | 見る場所 | 理由 |
|---|---|---|
| 0〜3か月 | 顔、首、胸、頭皮、おむつ周り | 乳児湿疹、よだれ、汗、うんち汚れが出やすい |
| 4〜6か月 | 首のしわ、手、口周り、肘膝 | よだれと手なめが増える |
| 7〜12か月 | 口周り、手足、膝、足首 | 離乳食、床遊び、ずりばいで刺激が増える |
| 1〜2歳 | 口周り、手、すね、肘膝、おむつ/下着周り | 外遊び、食べこぼし、汗、乾燥が増える |

### 保湿剤の使い分け

| 種類 | 特徴 | 向く場面 |
|---|---|---|
| ローション | のびやすい、水分が多い | 全身に広く塗りやすい |
| クリーム | ローションより保護感がある | 乾燥しやすい部位、冬 |
| ワセリン系 | 水分を閉じ込める、べたつく | 口周り、おむつ周り、強くこすれる場所 |

塗る量は「薄く気持ちだけ」では足りないことがあります。  
入浴後に水分を押し拭きし、乾き切る前に全身へ広げます。赤みが強い、じゅくじゅくする、かゆくて眠れない、保湿で改善しない時は、保湿剤を増やすだけでなく受診します。

### 受診した方がよい場合

- 湿疹が広がる
- じゅくじゅくする
- 眠れないほどかゆそう
- 保湿で改善しない
- 離乳食開始前に顔や体の湿疹が強い
- 食後にじんましんが出た

## 8-3. 歯のケア

歯のケアは、歯がたくさん生えてから始めるものではありません。  
最初の歯が見えた時点で、「口を触られる」「歯ブラシが入る」「寝る前に仕上げる」ことに慣れていきます。

日本小児歯科学会など4学会合同の提言では、歯が生えてから2歳までは、900〜1000ppmFのフッ化物配合歯磨剤を米粒程度、就寝前を含め1日2回使う方法が示されています<sup>[1](#ref-jpd-fluoride)</sup>。実際の商品選びや量に迷う場合は、歯科や健診で確認します。

### 用語補足

| 用語 | 意味 |
|---|---|
| 乳歯 | 子どもの歯。いずれ永久歯に生え替わるが、むし歯予防は重要 |
| 仕上げ磨き | 子どもが磨いた後、親が最後に磨くこと |
| フッ化物 | 歯の質を強くし、むし歯予防に使われる成分 |
| ppmF | 歯磨剤に含まれるフッ化物濃度の単位 |
| うがい不要量 | 少量の歯磨剤で、吐き出しやうがいが難しい年齢でも使いやすい量 |

### 0〜2歳の歯みがき

- [ ] 歯が生えたらガーゼや歯ブラシで慣らす
- [ ] 仕上げ磨きの姿勢に慣れる
- [ ] フッ化物配合歯磨剤は年齢に合う量を確認
- [ ] 寝る前の仕上げ磨きを習慣化
- [ ] 甘い飲み物をだらだら飲ませない
- [ ] 1歳半頃までに歯科相談を検討

### 月齢別：歯と口のケア

| 時期 | ケア | 目的 |
|---|---|---|
| 歯が生える前 | 口周りを拭く、口に触れられることに慣れる | 歯みがき前の準備 |
| 歯が生え始めたら | 小さな歯ブラシ、少量のフッ化物配合歯磨剤 | むし歯予防と習慣化 |
| 6〜12か月頃 | 寝る前の仕上げ磨きを固定 | 夜間の口の中を清潔にする |
| 1〜2歳 | 自分で持つ練習＋親の仕上げ磨き | 自立の芽を残しつつ磨き残しを減らす |

### 仕上げ磨きのコツ

- [ ] 毎回完璧に長時間やろうとしない
- [ ] 短く、同じ姿勢、同じ声かけで行う
- [ ] 上唇を少し持ち上げ、前歯の根元を見る
- [ ] 嫌がる時は、まず歯ブラシを口に入れる練習だけでもよい
- [ ] 歯ブラシを持たせる場合は、歩き回らせない
- [ ] 寝る前だけは親が仕上げる

歯みがきを嫌がるのは、親を困らせるためではありません。  
口の中は敏感で、仰向け、口を開ける、歯ブラシが当たる、唾液がたまるなど、子どもにとっては情報量が多い行為です。短時間で終わる、歌を決める、鏡を見る、親も一緒に磨くなど、予測できる流れにします。

### むし歯を防ぐ生活のポイント

- 甘い飲み物を哺乳瓶やマグでだらだら飲ませない
- 寝る前の授乳・ミルク後に歯が汚れる場合は歯科で相談する
- おやつの時間を決め、だらだら食べを避ける
- 仕上げ磨きとフッ化物を習慣にする
- 1歳半健診や歯科相談を使う

乳歯はいずれ抜けますが、乳歯のむし歯は痛み、食べにくさ、睡眠、永久歯への影響、歯科への恐怖につながることがあります。  
「まだ小さいから歯科は早い」ではなく、困る前に相談先を作るのが安全です。

## 8-4. 予防接種・健診

日本小児科学会は予防接種スケジュールを公開しています<sup>[2](#ref-jps-vaccine)</sup>。  
生後2か月から複数のワクチンが始まるため、退院後早めに小児科へ予約します。

### 予防接種は「予定表」ではなく「運用」で守る

予防接種は、親が毎回ゼロから調べ直すものではありません。  
小児科と一緒に、初回予約、同時接種、接種後の次回予約、体調不良時のキャッチアップまでを「運用」として固定します。

- 初回は生後2か月を目安に始める
- 生後1か月健診の前後で初回予約を取る
- 同時接種を前提に、小児科とスケジュールを組む
- 体調不良で延期したら、自己判断で放置せずキャッチアップを相談する<sup>[3](#ref-jpeds-catchup-2026)</sup>
- 母子健康手帳を毎回持参し、接種記録を確認する
- 接種後に次回予約を取る
- 保育園など集団生活に入る前は、未接種や延期分がないか確認する

同時接種は「一度に何本も打つのがかわいそう」と感じやすいところです。  
ただ、乳児期は短い期間に複数のワクチンを進める必要があります。接種を分けすぎると、完了まで時間がかかり、その間にワクチンで防げる病気から守れない期間が伸びます。日本小児科学会は、同時接種について、有効性への干渉や副反応頻度の上昇はなく、接種できるワクチン本数に原則制限はないという考え方を示しています<sup>[4](#ref-jpeds-simultaneous-vaccine)</sup>。実際の接種は、その日の体調と医師の判断に従います。

### チェックリスト

- [ ] 生後2か月前に小児科予約
- [ ] 母子手帳を毎回持参
- [ ] 接種予定表を冷蔵庫や共有アプリに入れる
- [ ] 予診票を事前記入
- [ ] 発熱・体調不良時の対応を小児科に確認
- [ ] 1か月健診、自治体健診の日程を確認
- [ ] 1歳6か月健診を忘れない
