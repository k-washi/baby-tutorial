# 第11章　親の運用設計：睡眠不足と不安への対策

## 11-1. 産後は「家事を回す」より「親が倒れない」

産後の家庭運営で最も大切なのは、母親だけに負担を集中させないことです。

赤ちゃんの世話は、授乳、おむつ、寝かしつけ、泣き対応、洗濯、哺乳瓶、予約、書類、買い物、予防接種、保育園調査が同時に来ます。  
これを「気合い」で回すと、睡眠不足の親が判断を誤ります。家庭内で、作業を見える化して分担します。

### 用語補足

| 用語 | 意味 |
|---|---|
| ワンオペ | 片方の親に育児・家事が集中している状態 |
| 夜勤 | 夜間授乳、おむつ、寝かしつけ、泣き対応を担当する時間帯 |
| メンタルロード | 予約、残量確認、予定管理など、見えにくい管理負担 |
| 最低運用 | 家事の完成度を落としても、家族の安全と睡眠を守る運用 |

### 産後前に決めること

- [ ] 夜間対応の交代ルール
- [ ] ミルクを使う場合の担当
- [ ] おむつ補充担当
- [ ] 洗濯担当
- [ ] 食事準備担当
- [ ] 買い物担当
- [ ] 来客対応ルール
- [ ] 祖父母サポートの範囲
- [ ] 「限界サイン」の共有

### 家庭内タスク表

| タスク | 主担当 | バックアップ | 頻度 |
|---|---|---|---|
| 夜間授乳・ミルク |  |  | 毎日 |
| おむつ補充 |  |  | 週1〜2回 |
| 哺乳瓶洗浄・消毒 |  |  | 毎日 |
| 洗濯 |  |  | 毎日〜隔日 |
| 食事準備 |  |  | 毎日 |
| ネットスーパー・買い物 |  |  | 週1〜2回 |
| 小児科予約 |  |  | 必要時 |
| 予防接種管理 |  |  | 月1程度 |
| 保育園・書類 |  |  | 時期ごと |
| 親の睡眠確保 |  |  | 毎日 |

ポイントは、手伝う人を決めることではなく、「誰が責任を持って完了確認するか」を決めることです。  
たとえば、おむつ替えを何回もしていても、在庫管理を誰も持っていなければ夜中に切れます。

### 夜間対応の設計

夜間対応は、眠い状態で毎回相談すると揉めます。  
産前または退院直後に、型を決めます。

| 方式 | 向く家庭 | 注意点 |
|---|---|---|
| 前半・後半交代 | ミルクや搾乳を使える、両親とも夜に対応できる | 母乳の場合、授乳担当の負担を別タスクで補う |
| 曜日交代 | 片方に出勤日が偏る家庭 | 完全休日を作りやすい |
| 授乳担当＋周辺担当 | 直接授乳中心の家庭 | おむつ、げっぷ、寝かしつけ、洗浄をもう片方が持つ |
| 外部サポート併用 | 祖父母、産後ケア、家事代行を使える家庭 | 依頼範囲と感染対策を事前に決める |

「授乳は代われないから何もできない」ではありません。  
水を渡す、上の子を見る、洗濯する、哺乳瓶を洗う、朝食を用意する、予約を取る、来客を断ることも育児です。

### 最低運用リスト

余裕がない日は、家事を全部やろうとしません。

1. 赤ちゃんの安全な睡眠
2. 授乳・ミルク・尿の確認
3. 親の睡眠
4. 食べられるものを食べる
5. 洗濯は赤ちゃん用品と最低限の下着だけ
6. 掃除は誤飲物、水回り、調乳周辺だけ
7. 返信、来客、写真整理、凝った料理は後回し

## 11-2. 心配性家庭の不安対策

### 不安を下げる仕組み

- [ ] 安全な寝床を作る
- [ ] ベビーモニターを使う
- [ ] 夜間確認の頻度を決める
- [ ] 小児科・救急相談先を見える場所に貼る
- [ ] 危険物チェックを週1回する
- [ ] パートナーと不安を共有する
- [ ] SNS検索しすぎない時間を作る

### 見守り機器より強いもの

1. 安全な寝床
2. 窒息物を置かない
3. 食事中に見守る
4. 誤飲物を床から消す
5. チャイルドシートを正しく使う
6. 親が寝る

## 11-3. 不安を「検索」ではなく「手順」に変える

不安が強い家庭ほど、検索量が増えます。  
検索は役に立つこともありますが、夜中にSNSや個人ブログを読み続けると、まれな事故や極端な体験談ばかりが目に入り、判断が乱れます。

### 不安ログ

| 書くこと | 例 |
|---|---|
| 何が心配か | ミルクが足りない気がする |
| 事実 | 尿6回、体温36.8度、機嫌は普通 |
| 追加で見る指標 | 次の授乳量、体重、便 |
| 相談先 | 小児科、助産師、#8000 |
| いつ相談するか | 尿が明らかに減る、飲めない、ぐったり |

不安は消そうとしなくてよいです。  
「何を見ればよいか」「どの条件で相談するか」に変換します。

### 相談の優先順位

| 状況 | 行動 |
|---|---|
| 呼吸が苦しそう、意識がぼんやり、けいれん、顔色が悪い | 救急要請を含めて急いで医療につなぐ |
| 発熱、哺乳不良、嘔吐、下痢、発疹、強い痛みがある | 小児科へ相談 |
| 夜間・休日で受診すべきか迷う | #8000など地域の相談窓口 |
| 予防接種、便秘、湿疹、離乳食、発達の継続相談 | かかりつけ小児科、健診、自治体相談 |
| 育児負担、睡眠不足、気分の落ち込み | 産科、助産師、自治体、家族、職場に相談 |

医療判断は、本書では確定しません。  
赤ちゃんの月齢、基礎疾患、出生状況、地域の医療体制で判断が変わるため、最終判断は小児科・産科・自治体の最新案内に従います。

## 11-4. 親のメンタルヘルス

産後は、睡眠不足、ホルモン変化、授乳、傷の痛み、責任感、孤立が重なります。  
気分が落ちる、涙が出る、眠れない、不安が止まらない、赤ちゃんをかわいいと思えない瞬間があること自体は珍しくありません。

ただし、次の状態が続く時は、早めに相談します。

- [ ] 眠れる状況でも眠れない
- [ ] 食べられない
- [ ] 涙が止まらない
- [ ] 強い不安や恐怖が続く
- [ ] 自分を責め続ける
- [ ] 赤ちゃんの世話ができないほどつらい
- [ ] 自分や赤ちゃんを傷つける考えが浮かぶ

これは親の弱さではなく、医療・支援につなぐべきサインです。  
産科、小児科、自治体の産後ケア、保健師、助産師、精神科・心療内科、家族に共有します。

### パートナーが見るサイン

- [ ] 表情が乏しい
- [ ] 眠れていない
- [ ] 食事が取れていない
- [ ] 「自分が悪い」と繰り返す
- [ ] 赤ちゃんの泣き声に強く怯える
- [ ] 受診や相談を自分で手配できない

「大丈夫？」だけでは足りないことがあります。  
具体的に、睡眠時間を作る、食事を出す、受診予約を取る、同行する、赤ちゃんを安全に見て休ませる、までを行動にします。

## 11-5. 週1回の家庭レビュー

育児は毎日変わります。  
週1回、10分だけ家庭レビューをすると、問題が大きくなる前に調整できます。

### 見る項目

- [ ] 赤ちゃんは飲めているか
- [ ] 尿・便は大きく変わっていないか
- [ ] 睡眠で困っていることは何か
- [ ] 親は連続して眠れているか
- [ ] 買い足すものは何か
- [ ] 予防接種・健診・保育園の締切は近いか
- [ ] 家事で捨てるものは何か
- [ ] 相談すべきことは何か

### 判断のルール

| 問い | 判断 |
|---|---|
| 毎日困っているか | 仕組みを変える |
| 週1回だけ困るか | 予備手段を作る |
| 体調・安全に関わるか | 相談先を決める |
| 親の睡眠を削っているか | 分担または外注を検討 |
| 検索で解決しないか | 小児科・自治体・専門職に聞く |

完璧な育児より、修正できる育児を目指します。
