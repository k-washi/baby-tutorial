# 第2章　出産前にやる手続き・仕事・お金

## 2-1. 出産前〜出産直後の手続き全体像

| 時期 | 手続き | 主な窓口 | 担当者のおすすめ |
|---|---|---|---|
| 妊娠判明後 | 妊娠届、母子健康手帳、妊婦健診補助券 | 自治体 | 妊婦本人 |
| 妊娠中 | 産休・育休・出生時育児休業の確認 | 勤務先 | 両親それぞれ |
| 妊娠中 | 出産育児一時金、出産手当金、育児休業給付の確認 | 健康保険・勤務先 | 主に勤務先 |
| 妊娠中 | 小児科探し、予防接種予約方針 | 小児科 | どちらか |
| 出生後14日以内 | 出生届 | 市区町村 | 父/母/同居者など |
| 出生後すぐ | 健康保険加入 | 勤務先または国保窓口 | 父母どちらか |
| 出生後すぐ | 乳幼児医療費助成 | 自治体 | 父母どちらか |
| 出生後すぐ | 児童手当 | 自治体 | 所得・扶養状況で確認 |
| 生後2か月前 | 予防接種予約 | 小児科 | 予約担当を固定 |

出生届は、出生の日から14日以内に届け出る必要があります<sup>[1](#ref-moj-birth)</sup>。児童手当は自治体への申請が必要で、制度内容は改正されることがあるため、出産前に住んでいる自治体のページを確認しておきます<sup>[2](#ref-cfa-childallowance)</sup>。

## 2-2. 仕事関連：産休・育休・出生時育児休業

### 出産前に確認すること

- [ ] 産前産後休業の開始日
- [ ] 育児休業の取得予定期間
- [ ] 父親の出生時育児休業、いわゆる「産後パパ育休」の取得可否
- [ ] 育児休業給付の申請フロー
- [ ] 2025年開始の出生後休業支援給付の対象になるか
- [ ] 社会保険料免除の扱い
- [ ] 賞与月・月末をまたぐ休業の影響
- [ ] 復職予定時期
- [ ] 保育園申請のスケジュール
- [ ] 会社への提出書類の締切

厚生労働省は、育児休業給付や出生後休業支援給付に関する情報を公開しています。制度は改正があるため、勤務先の人事とハローワーク系の最新資料を確認するのが安全です<sup>[3](#ref-mhlw-leave)</sup>。

### 実践Tips

- 産後は睡眠不足で書類処理能力が落ちます。
- 申請書類の記入例、提出先、期限を妊娠中にメモしておきます。
- スマホのメモではなく、夫婦で見られる共有ドキュメントにします。
- 「出生届後にやること」を1枚にまとめ、母子手帳ケースに入れておくと楽です。

## 2-3. 出産費用と産院選び

厚生労働省の「出産なび」では、分娩を取り扱う施設の情報や費用の目安を検索できます<sup>[4](#ref-mhlw-delivery)</sup>。  
出産費用は地域差・施設差・個室・無痛分娩・休日夜間・帝王切開などで大きく変わります。

### 出産前チェック

- [ ] 分娩予約が完了している
- [ ] 出産費用の概算を聞いた
- [ ] 直接支払制度の扱いを確認した
- [ ] 無痛分娩の可否・追加費用を確認した
- [ ] 休日・夜間加算を確認した
- [ ] 帝王切開時の支払い・保険適用を確認した
- [ ] 入院中に必要な持ち物リストを受け取った
- [ ] 退院時の赤ちゃんの移動方法を決めた
- [ ] 車ならチャイルドシートを装着済み

## 2-4. 小児科・予防接種・保育園は妊娠中に先回りする

小児科、予防接種、保育園申請は、産後に調べ始めると遅れやすい領域です。  
赤ちゃんが生まれると、授乳、睡眠不足、出生届や健康保険の手続きで、親の調査力は大きく落ちます。妊娠中に「どこへ電話するか」「いつ予約するか」「何をカレンダーに入れるか」まで決めておくと、産後の不安がかなり下がります。

- 小児科候補は妊娠中に2〜3院リスト化する
- 生後2か月の初回予防接種は、生後1か月健診の前後で予約する
- 厚生労働省は、生後2か月から始まるワクチンとしてロタ、5種混合、小児用肺炎球菌、B型肝炎などを案内している<sup>[5](#ref-mhlw-vaccine-2months)</sup>
- 4月保育園入園は、前年秋に申請が始まる自治体が多い
- 保育園は自治体・園ごとに締切、受入月齢、必要書類、選考方法が異なる
- 認可保育園などの保育利用では、市区町村の認定・利用調整が中心になる<sup>[6](#ref-cfa-hoiku-system)</sup>
- 産前に、予防接種と保育園申請の締切を共有カレンダーへ入れる
