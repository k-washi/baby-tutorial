# 第12章　小児科探し・予防接種予約・保育園申請

この章は、妊娠後期から0歳前半にかけて、親が先回りして仕組み化しておく章です。  
医療、予防接種、保育園申請は、どれも「直前に調べる」では遅れやすい領域です。医療判断、接種可否、保育園申請の締切や必要書類は、小児科、産科、お住まいの市区町村、園の最新案内に従ってください。

---

## 12-1. この章の結論

小児科探しは、産後に慌てて始めるより、妊娠後期から生後1か月健診までに候補を2〜3院決めておきます。  
赤ちゃんは生後2か月から予防接種が本格的に始まります。生後1か月健診の前後で初回予約を取り、以後は「接種したら次回予約を取る」を家庭の運用にします。

予防接種は、同時接種を前提に、遅らせず、小児科と一緒にスケジュールを組みます。厚生労働省は生後2か月から始まるワクチンとして、ロタ、5種混合、小児用肺炎球菌、B型肝炎などを案内しています<sup>[1](#ref-mhlw-vaccine-2months)</sup>。制度やスケジュールは変わることがあるため、2026年6月時点でも、最終判断は厚生労働省、自治体、小児科の最新案内で確認します<sup>[2](#ref-mhlw-vaccine-2026)</sup>。

保育園申請は自治体差が大きい領域です。4月入園は前年秋に一次申込が始まることが多いため、妊娠中から逆算します。保育園は「入りたい園」だけでなく、「通える園」「受入月齢が合う園」「復職日と慣らし保育が合う園」で考えます。

## 12-2. 小児科探し：いつ、何を基準に選ぶか

### いつ探すか

| 時期 | やること | 理由 |
|---|---|---|
| 妊娠後期 | 自宅周辺の小児科を3〜5院リストアップ | 産後は調べる余裕が少ない |
| 出産前 | 予防接種・乳幼児健診を扱うか確認 | 生後2か月から接種が始まる |
| 生後1か月健診前後 | 第一候補の小児科で初回予防接種を予約 | 予約枠が埋まることがある |
| 生後2か月 | 初回接種 | ロタ、5種混合、肺炎球菌、B型肝炎など |
| 生後2か月以降 | 接種後に次回予約を入れる | スケジュール漏れを防ぐ |

小児科は「家から近い」だけで選ぶと、予防接種や健診の運用で困ることがあります。  
一方で、専門性だけで遠い医院を選ぶと、発熱時、雨の日、抱っこ移動の日に通いにくくなります。0〜2歳の小児科は、日常の体調不良、予防接種、乳幼児健診、湿疹、便秘、哺乳不良、発達の相談まで何度も通う場所です。継続して通えること自体が、安全性と安心感につながります。

東京都こども医療ガイドは、定期予防接種を中心に、どの年齢でどのワクチンを受けるか、かかりつけ医等と相談してスケジュールを作るとよいと案内しています<sup>[3](#ref-tokyo-vaccine-guide)</sup>。つまり、小児科探しは「風邪をひいたら行く場所探し」だけでなく、「予防接種と健診を一緒に管理してくれる相手探し」です。

### 小児科選びのチェックリスト

- [ ] 自宅から通いやすい。発熱時・雨の日・抱っこ移動でも行ける
- [ ] 予防接種と乳幼児健診を扱っている
- [ ] 生後2か月の初回ワクチンを予約できる
- [ ] Web予約・アプリ予約に対応している
- [ ] 予防接種・健診の時間帯が、一般診療と分かれている、または感染対策が明確
- [ ] 同時接種に前向きで、スケジュールを組んでくれる
- [ ] ロタワクチンの種類を管理し、同じ種類で完了できるようにしてくれる
- [ ] 発熱・咳・湿疹・便秘・哺乳不良など、日常の相談がしやすい
- [ ] 発達相談、離乳食相談、皮膚トラブル相談に対応してくれる
- [ ] 予約外・当日診療のルールが明確
- [ ] 休診日・夜間・休日に困った時の案内がある
- [ ] 近隣の救急病院・紹介先との連携がある
- [ ] 薬局が近い、または処方薬を受け取りやすい
- [ ] ベビーカーで入れる、エレベーターがある、駐輪場/駐車場が使える
- [ ] 受付・医師・看護師の説明がわかりやすい
- [ ] 親の不安を否定せず、質問しやすい

### 小児科は「1院だけ」にしない

第一候補は、かかりつけとして、予防接種、健診、普段の体調不良を見てもらう医院です。  
第二候補は、第一候補が休診日、予約満了、急な発熱で受診できない時のバックアップです。夜間・休日は地域の小児救急、救急相談窓口、#8000を別枠で確認します。厚生労働省は、#8000を休日・夜間の子どもの急な症状について相談できる日本全国統一の短縮番号として案内しています<sup>[4](#ref-mhlw-8000)</sup>。

皮膚トラブルが強い子は小児科に加えて皮膚科、耳・鼻の症状が多い子は耳鼻科も候補に入れます。便秘、アレルギー、発達など継続相談が必要な場合は、通院しやすさと質問しやすさを優先します。

## 12-3. 予防接種予約方針

予防接種は、親が毎回ゼロから考えるものではなく、小児科と一緒に「次回予約を取り続ける」仕組みにします。

### 基本方針

1. 生後2か月になったらすぐ開始する
2. 予約は生後1か月健診前後に入れる
3. 同時接種を前提にする
4. 接種後に次回予約を取る
5. 母子健康手帳に記録する
6. 体調不良で遅れたら、かかりつけ医にキャッチアップを組んでもらう<sup>[5](#ref-jpeds-catchup-2026)</sup>
7. 保育園など集団生活に入る予定がある場合は、特に遅らせない

### 生後2か月から始まる主なワクチン

以下は2026年6月時点での一般的な整理です。細かい接種間隔、対象年齢、定期・任意の扱い、同時接種の可否は、必ず最新の厚生労働省、自治体、小児科の案内に従ってください。日本小児科学会は2026年4月1日版の推奨スケジュールを公開しています<sup>[6](#ref-jpeds-schedule-2026)</sup>。

| 時期 | 主なワクチン | 方針 |
|---|---|---|
| 妊娠28〜36週頃 | RSウイルス母子免疫ワクチン | 産科で確認。2026年版スケジュールでは妊娠28週0日〜36週6日の妊婦に位置づけられている |
| 生後2か月 | ロタ、5種混合、小児用肺炎球菌、B型肝炎 | 初回接種。ここを遅らせない |
| 生後3か月 | 2回目接種群 | 初回後に次回予約を済ませておく |
| 生後4か月 | 3回目接種群 | 接種間隔を小児科で管理 |
| 生後5か月頃 | BCG | 自治体・小児科方式に従う |
| 生後7〜8か月頃 | B型肝炎3回目など | 抜け漏れ注意 |
| 1歳 | MR、水痘、肺炎球菌追加、5種混合追加など | 1歳になったら早めに予約 |
| 毎年秋冬 | インフルエンザ | 対象年齢・接種回数を小児科で確認 |
| 1歳以降 | おたふくかぜなど任意接種 | 小児科で相談 |

### 同時接種について

乳児期は、短期間に複数回の接種が必要になります。  
日本小児科学会は、同時接種について、複数ワクチンを同時に接種しても有効性への干渉はなく、副反応頻度が上がることはなく、接種できる本数に原則制限はないという考え方を示しています<sup>[7](#ref-jpeds-simultaneous-vaccine)</sup>。

同時接種の利点は、接種率が上がること、ワクチンで防げる病気から早期に守れること、保護者の時間的負担が減ることです。  
「本数が多くてかわいそう」という気持ちは自然です。ただ、接種を分けすぎて遅れることにもリスクがあります。実際には、医師がその日の体調、月齢、過去の接種歴、ワクチンの種類を見て判断します。

### 接種当日の持ち物チェックリスト

- [ ] 母子健康手帳
- [ ] 予診票
- [ ] マイナ保険証または健康保険証情報
- [ ] 乳幼児医療証
- [ ] 診察券
- [ ] おむつ
- [ ] おしりふき
- [ ] 着替え
- [ ] ガーゼ・タオル
- [ ] ミルク・哺乳瓶または授乳用品
- [ ] 体温を測っておく
- [ ] 前日からの発熱・下痢・嘔吐・発疹・機嫌の悪さをメモ
- [ ] 質問メモ

### 接種後の行動

- 接種後30分程度は医療機関で様子を見る
- 当日は激しい運動を避ける
- 体調に問題がなければ入浴は可能とされることが多いが、医師の指示に従う
- 発熱、強い腫れ、ぐったり、呼吸が苦しそう、じんましん、嘔吐などがあれば医療機関へ相談
- 母子健康手帳の記録を確認する
- 帰宅前または当日中に次回予約を取る

### 予防接種スケジュール運用チェックリスト

- [ ] 出生後、自治体から予診票が届く時期を確認した
- [ ] 生後1か月健診前後で小児科を決めた
- [ ] 生後2か月の初回接種予約を取った
- [ ] ロタワクチンの種類を確認した
- [ ] 小児科に同時接種方針を確認した
- [ ] 接種後に次回予約を取る運用にした
- [ ] 母子健康手帳の接種記録を毎回確認している
- [ ] 体調不良で延期した時はキャッチアップ予約を取る
- [ ] 保育園入園予定月から逆算して未接種がないか確認した
- [ ] 任意接種について小児科に相談した

## 12-4. 保育園申請の基本構造

認可保育園、認定こども園の保育利用、地域型保育などは、市区町村が窓口になります。  
0〜2歳で保育を必要とする場合は原則として3号認定、3〜5歳で保育を必要とする場合は原則として2号認定です。こども家庭庁は、2号・3号認定では市町村が保育の必要性を認め、申請者の希望や保育所などの状況、保育の必要性の程度を踏まえて利用調整を行う流れを示しています<sup>[8](#ref-cfa-hoiku-system)</sup>。厚生労働省の保育所等の利用調整に関する通知でも、市町村が利用調整を行う前提が示されています<sup>[9](#ref-mhlw-hoiku-adjustment)</sup>。

園が直接「あなたに内定を出す」というより、認可保育では市区町村の利用調整が中心です。認証保育所、認可外保育施設、企業主導型保育などは、申込先、契約方式、補助金が異なるため、認可保育とは別枠で確認します。

### 保育園の種類

| 種類 | 申込先 | 特徴 |
|---|---|---|
| 認可保育園 | 市区町村 | 保育の必要性認定と利用調整がある |
| 認定こども園 保育利用 | 市区町村中心 | 教育と保育の機能。認定区分に注意 |
| 小規模保育 | 市区町村中心 | 主に0〜2歳。3歳以降の連携園・転園を確認 |
| 家庭的保育 | 市区町村中心 | 少人数。保育者との相性確認が重要 |
| 認証保育所/認可外 | 園に直接の場合が多い | 自治体補助、料金、空き確認が必要 |
| 企業主導型 | 園・事業者 | 地域枠/従業員枠、料金、補助条件を確認 |
| 一時保育・病児保育 | 自治体/施設 | 復職前後や緊急時の保険になる |

## 12-5. 保育園申請スケジュール

### 4月入園を狙う場合

4月入園は、前年秋から動きます。  
申請の直前に園見学を始めると、見学枠、就労証明書、希望順位の検討が間に合わないことがあります。

| 時期 | やること |
|---|---|
| 妊娠中〜産後早期 | 住んでいる自治体の保育園案内を読む |
| 4〜6月 | 保育園の種類、通園可能範囲、復職時期を整理 |
| 6〜8月 | 園見学の予約、候補園リスト作成 |
| 8〜10月 | 就労証明書など必要書類を確認 |
| 9〜11月 | 募集要項・申請書類の配布開始を確認 |
| 10〜12月 | 4月入園一次申込 |
| 1〜2月 | 一次結果発表 |
| 2〜3月 | 二次申込、内定後面談、健康診断、入園説明会 |
| 3月 | 持ち物準備、慣らし保育調整、復職日調整 |
| 4月 | 入園、慣らし保育、復職 |

### 東京の自治体例

以下は東京都内自治体の例です。自治体によって日程は毎年変わるため、ここは「東京都の全自治体に共通する日程」ではありません。

- 新宿区の令和8年4月入園は、1次締切が令和7年12月5日、結果発表予定が令和8年2月13日、2次締切が令和8年2月3日、結果発表予定が令和8年2月27日です<sup>[10](#ref-shinjuku-hoiku-2026)</sup>
- 目黒区は令和8年度案内を令和7年10月1日以降に順次配布し、令和7年度案内では2月・3月入所はないとしています<sup>[11](#ref-meguro-hoiku-2026)</sup>
- 中央区の令和8年度入所申込みでは、令和8年4月入所の第1回受付が令和7年11月4日〜28日、第2回受付が令和8年2月2日〜13日で、令和9年3月入所は取り扱わないとされています<sup>[12](#ref-chuo-hoiku-2026)</sup>

つまり、「4月入園は前年秋」「年度途中は入園希望月の前月または前々月締切」「2〜3月入園を扱わない自治体もある」という前提で、お住まいの市区町村のページを必ず確認します。

### 年度途中入園の場合

年度途中入園は、空きがあれば毎月選考される自治体が多いです。  
ただし、0歳・1歳の空きは地域差が大きく、途中入園が難しいこともあります。締切は「入園希望月の前月上旬」「前月末」「2か月前」など自治体差があります。

途中入園では、希望園の空き状況を毎月確認します。難しい場合は、認可外、認証、企業主導型、一時保育、育休延長などの代替策を同時に検討します。

### 0歳4月入園・1歳4月入園の考え方

| 戦略 | メリット | 注意点 |
|---|---|---|
| 0歳4月入園 | 0歳枠がある園では入りやすい場合がある。復職が早い家庭に合う | 受入月齢、産後の親子の負担、授乳、慣らし保育に注意 |
| 1歳4月入園 | 育休を長めに取れる。子どもの生活リズムが安定しやすい | 地域によっては1歳枠が激戦。0歳からの持ち上がりで新規枠が少ない場合がある |
| 年度途中入園 | 復職月に合わせやすい | 空きがなければ入れない。候補園を広げる必要がある |
| 認可外・認証併願 | 復職の保険になる | 費用、補助、通園距離、安全性、契約条件を確認 |

### 受入月齢を必ず確認する

0歳児クラスでも、園によって受入開始月齢が違います。  
生後57日、3か月、6か月など園ごとに異なることがあります。新宿区も、0歳児クラスの受入月齢は保育園によって異なると案内しています<sup>[13](#ref-shinjuku-hoiku-age)</sup>。

入園月の1日時点で受入月齢を満たしているかを確認します。出産予定日が冬〜早春の場合、4月0歳入園に間に合うかは自治体・園のルール次第です。受入月齢、クラス年齢、出生日基準、出生前の仮申込可否を自治体に確認します。

## 12-7. 保育園申請の必要書類チェックリスト

自治体差があるため、以下は一般例です。書類名、様式、提出方法、締切、電子申請の締切時刻は、お住まいの市区町村の最新案内を確認します。

- [ ] 教育・保育給付認定申請書
- [ ] 保育施設利用申込書
- [ ] 児童の状況申告書
- [ ] 就労証明書
- [ ] 復職証明書、または復職予定申立書
- [ ] 育児休業取得証明書
- [ ] 母子健康手帳の写し、出生予定がわかる書類
- [ ] 住民税関係書類、課税証明書が必要な場合
- [ ] マイナンバー確認書類
- [ ] 本人確認書類
- [ ] ひとり親、疾病、介護、障害、求職、就学などの該当証明
- [ ] 転入予定の場合の賃貸契約書・売買契約書等
- [ ] 認可外保育施設利用証明書が必要な場合
- [ ] 不足書類締切日を確認した
- [ ] 郵送の場合は必着/消印有効の違いを確認した
- [ ] 電子申請の場合は締切時刻を確認した

## 12-8. 家庭別スケジュールテンプレート

### 妊娠中に4月入園を検討する家庭

- [ ] 出産予定日から、4月1日時点の月齢を確認
- [ ] 0歳4月入園が可能な受入月齢か確認
- [ ] 産前に園見学を済ませる
- [ ] 募集要項の公開時期をカレンダー登録
- [ ] 就労証明書を勤務先に依頼
- [ ] 申込締切の2週間前までに書類を揃える
- [ ] 第一希望だけでなく、通える園を広めに書く
- [ ] 認可外・認証・企業主導型も併願検討

### 1歳4月入園を狙う家庭

- [ ] 前年春〜夏に自治体の指数表を読む
- [ ] 0歳児からの持ち上がりで1歳新規枠が少ない園を把握
- [ ] 1歳児から受入開始の園を確認
- [ ] 小規模保育・認可外も含めて候補を作る
- [ ] 育休延長条件と職場復帰期限を確認
- [ ] 申請締切の2週間前までに書類を揃える

### 年度途中入園を狙う家庭

- [ ] 入園希望月の締切日を確認
- [ ] 空き状況の公開日を確認
- [ ] 毎月、空き状況を確認
- [ ] 希望園変更の締切を確認
- [ ] 認可外・一時保育・病児保育を同時に調べる
- [ ] 内定後の慣らし保育期間を復職日と調整

## 12-9. 親がやりがちな失敗と対策

| 失敗 | 起きること | 対策 |
|---|---|---|
| 小児科を産後に探す | 生後2か月接種の予約が遅れる | 妊娠後期に候補を作る |
| 予防接種を1本ずつ分ける | 完了が遅れる | 医師と相談し同時接種を基本にする |
| 接種後に次回予約しない | 抜け漏れが起きる | 帰宅前に次回予約 |
| 母子手帳を忘れる | 接種記録が残らない | 持ち物セットに常備 |
| 第一希望園だけ見る | 落ちた時に詰む | 通える園を広めに見学 |
| 受入月齢を見落とす | 申込できない園を書く | 0歳は必ず受入月齢確認 |
| 申請締切を見落とす | 次月/次年度まで待つことになる | 自治体締切をカレンダー登録 |
| 就労証明書を直前依頼 | 勤務先処理が間に合わない | 1か月前に依頼 |
| 慣らし保育を考えない | 復職初日に働けない | 復職日は余裕を持つ |
| 小規模保育の3歳以降を確認しない | 2歳後に再保活 | 連携園・卒園後加点を確認 |
