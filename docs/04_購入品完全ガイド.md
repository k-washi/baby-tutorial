# 第4章　購入品完全ガイド

## 4-1. 購入の基本方針

赤ちゃん用品は、全部を出産前にそろえる必要はありません。

### 出産前に買うもの

- 退院直後に必要
- 安全に関わる
- 毎日使う
- 産後すぐ買いに行くのが大変

### 産後に買ってよいもの

- 赤ちゃんの体型・好みに左右される
- 家の動線に左右される
- 月齢が進まないと使わない
- 高額でレンタルや試用ができる

## 4-3. ベビーカー：A型を買うべきか

### 結論

徒歩移動・電車移動・買い物が多いなら、A型を買う価値があります。  
車中心で、しばらく外出が少ない家庭なら、抱っこ紐で様子見して、7か月頃からB型を検討する選択もあります。

### アップリカ ラクーナクッション AH A型の判断

アップリカ ラクーナクッション AHは、公式情報では生後1か月〜36か月、体重15kg以下、リクライニング123〜166度、重さ5.6kgのA形SG基準適合品です<sup>[1](#ref-aprica)</sup>。  
生後早期から外出する家庭には候補になります。A形/B形などの区分はSG基準の対象月齢や用途にも関わるため、公式仕様とSG基準の表示を確認して選びます<sup>[2](#ref-sg-stroller)</sup>。

### A型が向く家庭

- [ ] 徒歩で小児科・買い物へ行く
- [ ] 産後の腰や肩に不安がある
- [ ] 電車移動がある
- [ ] 荷物を載せたい
- [ ] 生後1〜6か月でも散歩・外出予定がある
- [ ] 玄関やエレベーターに置ける

### A型を見送ってもよい家庭

- [ ] 車移動が中心
- [ ] 玄関が狭い
- [ ] 外出は抱っこ紐で足りそう
- [ ] 7か月以降に軽いB型を買いたい
- [ ] レンタルで試したい

### 買う前の実測チェック

- [ ] 玄関に置ける
- [ ] たたんだ時のサイズが収納に合う
- [ ] エレベーターに入る
- [ ] 車のトランクに入る
- [ ] よく使う道の段差を押せる
- [ ] 片手で畳める
- [ ] 片手で持てる重さ
- [ ] 対面・背面の切替が必要か確認した
- [ ] 荷物カゴに普段の買い物量が入る
- [ ] 店頭で実際に押した

### Tips

- 「軽い」は正義ですが、軽すぎると段差で不安定に感じる家庭もあります。
- 片手開閉は、実際に赤ちゃんを抱えた想定で試します。
- 雨の日に使うならレインカバーを同時に検討します。
- 玄関に置けないベビーカーは、使用頻度が下がります。

## 4-4. チャイルドシート

車に乗せるなら、退院時から必要です。  
日本では6歳未満の幼児にチャイルドシート使用義務があります。警察庁は、チャイルドシート不使用時の致死率が適正使用時より高いことを公表しています<sup>[3](#ref-npa-childseat)</sup>。後ろ向き・前向き・ジュニアシートへの移行は、商品説明書と車種適合、JAF等の安全情報を確認します<sup>[4](#ref-jaf-childseat)</sup>。

### 選ぶポイント

- [ ] 新生児から使える
- [ ] 車種適合を確認した
- [ ] ISOFIX対応か確認した
- [ ] 後ろ向きで使える期間が長い
- [ ] 装着ミスしにくい
- [ ] ベルト調整が簡単
- [ ] カバーが洗える
- [ ] 回転式が必要か確認した

### Tips

- 退院日に初めて装着すると焦ります。妊娠中に車へ取り付けます。
- 回転式は乗せ降ろしが楽ですが重いです。
- 複数台の車で使うなら、付け替えやすさを重視します。

## 4-5. 抱っこ紐

### 結論

メイン抱っこ紐は、夫婦で試着して決めます。  
候補は次のように考えます。

| 商品タイプ | 向く家庭 |
|---|---|
| エルゴベビー OMNI Breeze系 | 長時間抱っこ、腰肩負担を減らしたい、夫婦兼用 |
| ベビービョルン HARMONY系 | 新生児期の装着しやすさ、前側バックル重視 |
| コニー・スモルビ等の布系 | 家の中、短時間、セカンド抱っこ紐 |
| ヒップシート | 腰すわり以降、短時間の乗せ降ろし |

エルゴベビーOMNI Breezeは、公式情報で3.2kg〜20.4kg、複数の抱き方に対応する設計です<sup>[5](#ref-ergo)</sup>。ベビービョルンHARMONYも新生児期から使えるタイプとして販売されています<sup>[6](#ref-babybjorn)</sup>。

### 買う前チェック

- [ ] 新生児から使える
- [ ] 首すわり前の支えがある
- [ ] 股関節が自然なM字になる
- [ ] 夫婦どちらも装着できる
- [ ] 腰ベルトが痛くない
- [ ] 肩ベルトが食い込まない
- [ ] 赤ちゃんの顔が見える
- [ ] 顎が胸に押し付けられない
- [ ] 洗濯できる
- [ ] 夏の蒸れ対策がある

### Tips

- 新生児期は「装着の簡単さ」が重要です。
- 長時間外出が多いなら腰ベルトの安定性を重視します。
- 家の中で寝かしつけに使うなら、軽いセカンド抱っこ紐が便利です。
- 抱っこ紐で寝たら、長時間そのままにせず安全な寝床に移します。

## 4-6. おむつ

### 結論

新生児期はテープタイプから始めます。  
寝返り・ずりばいで替えづらくなったらパンツタイプへ移行します。

### テープとパンツの使い分け

| 状態 | おすすめ |
|---|---|
| 新生児〜寝返り前 | テープ |
| 足バタバタが強い | テープ＋パンツ併用 |
| 寝返りで逃げる | パンツ |
| ずりばい・ハイハイ | パンツ |
| うんちが多い時 | テープが処理しやすいこともある |
| 夜間漏れ | サイズアップ、夜だけパンツ、吸収量重視 |

Moonyは、赤ちゃんの動きが活発になり、おむつ替えが難しくなったらパンツタイプへの切替を案内しています<sup>[7](#ref-moony)</sup>。Merriesの情報でも、寝返り期以降にパンツ移行が増える傾向が示されています<sup>[8](#ref-merries)</sup>。

### ムーニー マシュマロ肌ごこちを検討する場合

第一候補にしてよいですが、最初から大量買いは避けます。  
理由は、赤ちゃんの体型、肌との相性、太もも周り、うんちのゆるさで漏れやすさが変わるからです。

### おむつ購入チェック

- [ ] 新生児サイズを1〜2パックだけ買う
- [ ] 産院で使うメーカーを確認する
- [ ] 太もも跡が強ければサイズアップを検討
- [ ] 背中漏れが多ければサイズ・当て方を見直す
- [ ] 夜間漏れは夜だけ別メーカーも試す
- [ ] 肌荒れ時はおむつだけでなく拭き方・保湿も見直す

### Tips

- 新生児サイズはすぐサイズアウトします。
- 漏れは「メーカーが悪い」より「体型との相性」のことが多いです。
- 同じSサイズでもメーカーで形が違います。
- パンツは横を破って外せます。
- うんち処理は、パンツよりテープの方が楽な場面があります。

## 4-7. おしりふき

### Dr.Heart薬用おしりふき：ノーマルと厚手/大判

| タイプ | 向く場面 |
|---|---|
| ノーマル | おしっこ、軽いうんち、日常使い |
| 厚手・大判 | 新生児のゆるいうんち、外出、肌荒れ時、少ない枚数で拭きたい時 |

Dr.Heartのノーマルタイプは175×130mm、薬用成分アラントイン配合などの仕様で販売されています。大判タイプは20cm×18cmのような大きめ仕様の商品があります<sup>[9](#ref-drheart)</sup>。

### 買い方

- [ ] 家用：ノーマルを多め
- [ ] 外出用：厚手・大判
- [ ] うんち用：厚手・大判
- [ ] 肌荒れ時：こすらず、ぬるま湯洗い＋保護剤
- [ ] 乾きにくいフタを用意

### Tips

- 新生児のうんちはゆるいので、厚手の方が結果的に使用枚数が少ないことがあります。
- おしりが赤い時は、拭く回数を減らす工夫が必要です。
- こするより、押さえて取る、ぬるま湯で流す、保護する、が基本です。

## 4-8. 哺乳瓶・粉ミルク

### 哺乳瓶容量の結論

最初は160mlを1〜2本。  
3〜4か月以降、1回量が増えたら240mlを追加します。

Pigeonは、はじめての哺乳びんとして160mlを案内し、240mlは月齢が進んだ後の選択肢として示しています<sup>[10](#ref-pigeon-bottle)</sup>。

### 本数の目安

| 授乳方針 | 出産前の目安 |
|---|---|
| 完母予定 | 160ml 1本 |
| 混合予定 | 160ml 2本 |
| 完ミ予定 | 160ml 3本以上＋必要に応じて240ml |
| 夜間・外出重視 | キューブ/液体ミルクも検討 |

### 粉ミルクの選び方

- [ ] 産院の銘柄を確認
- [ ] 最初は少量
- [ ] 缶は使い切れる量か確認
- [ ] キューブは夜間・外出に便利
- [ ] 液体ミルクは災害備蓄に便利
- [ ] アレルギーや特殊ミルクは医師に相談

明治ほほえみのキューブタイプは、スプーン計量不要で扱いやすい点が特徴です<sup>[11](#ref-meiji-cube)</sup>。液体ミルクは調乳不要で、外出や災害時の選択肢になります<sup>[12](#ref-meiji-liquid)</sup>。

### Tips

- 完母予定でも哺乳瓶1本はあると安心です。
- 新生児期から240mlだけでも使えますが、量が少ない時期は160mlの方が扱いやすいです。
- 乳首サイズは月齢に合わせます。飲むのが遅い/むせる/疲れる時はサイズを見直します。
- ミルクは「高いほどよい」ではなく、赤ちゃんが飲める・家庭が続けられる・衛生的に扱えることが大切です。

## 4-10. ベビーソープ・ローション

### 結論

泡タイプのベビーソープと、毎日使いやすい保湿剤を用意します。  
肌トラブルが強い場合は、保湿剤を増やすだけで粘らず小児科・皮膚科へ相談します。

### 選び方

| 品目 | 重視する点 |
|---|---|
| ベビーソープ | 泡タイプ、無香料、低刺激、洗い流しやすい |
| ローション | 無香料、伸びがよい、毎日続けやすい |
| クリーム | 乾燥が強い部位用 |
| ワセリン | よだれ・おむつかぶれの保護 |

### 根拠

新生児期からの保湿がアトピー性皮膚炎リスクを下げる可能性を示した日本の研究があります<sup>[13](#ref-skin-horimukai)</sup>。一方で、より大規模な研究では、毎日の保湿だけで湿疹予防を確実に達成できるとは言い切れない結果もあります<sup>[14](#ref-skin-beep)</sup>。  
したがって実践上は、「保湿は皮膚バリアを守る日常ケアとして大切。ただし、アトピーや食物アレルギーを必ず予防する魔法ではない」と理解するのが正確です。

### Tips

- 入浴後はなるべく早めに保湿します。
- 湿疹がある時は、食物制限より先に皮膚治療を相談します。
- よだれかぶれは、保湿＋ワセリンで保護するのが実用的です。
- 香りが強いものは避ける方が無難です。

## 4-11. ベビーチェア

### 買う時期

離乳食開始の5〜6か月頃で十分です。

### 選ぶポイント

- [ ] 足がつく
- [ ] 姿勢が安定する
- [ ] ベルトがある
- [ ] 掃除しやすい
- [ ] テーブルとの高さが合う
- [ ] 抜け出しにくい
- [ ] 長く使うか、短期で割り切るか決める

### Tips

- 足がぶらぶらすると食べにくい子がいます。
- 離乳食の後は毎回汚れるので、掃除しやすさは重要です。
- ローチェアかハイチェアかは、家族の食卓スタイルに合わせます。

## 4-12. バウンサー

### 結論

必須ではありません。  
使うなら「短時間の待機場所」です。寝床ではありません。

### 使ってよい場面

- [ ] 親がトイレに行く短時間
- [ ] 家事中に目の届く場所へ置く
- [ ] 上の子・ペットから一時的に距離を取る
- [ ] 食後すぐの短時間の待機

### 使わない場面

- [ ] 夜の睡眠
- [ ] 昼寝のメイン寝床
- [ ] 親が寝てしまう時
- [ ] 長時間の固定
- [ ] ベルトなし

### Tips

- 寝たら安全な寝床へ移します。
- 長時間使うと床で動く時間が減ります。
- 「泣き止むから常用」ではなく、親を守る補助具として使います。

## 4-13. ベビーバス・沐浴マット

### 結論

新生児期だけなら空気式ベビーバスが扱いやすいです。  
ワンオペ入浴が多い家庭は、沐浴マットやバスマットも便利です。

| タイプ | 向く家庭 |
|---|---|
| 空気式ベビーバス | 省スペース、短期使用 |
| プラスチック浴槽 | 安定感重視 |
| 洗面台マット | 洗面台が広く清潔にできる |
| 沐浴マット | 洗い場で寝かせて洗いたい |
| 折りたたみバス | 収納性と安定性のバランス |

### 買う前チェック

- [ ] 置く場所に入る
- [ ] 排水しやすい
- [ ] 乾かしやすい
- [ ] 親の腰が痛くなりにくい
- [ ] 滑りにくい
- [ ] 1か月後も使うか想定した

## 4-14. 買うか迷いやすい補助用品

鼻吸い器、ベビーピンセット、除菌乾燥機、おむつ専用ゴミ箱、搾乳器、指挟み防止グッズは、「全家庭の必須品」ではありません。  
ただし、生活条件によってはかなり助かります。判断軸は、次の3つです。

1. 赤ちゃんの安全・衛生に関わるか
2. 親の睡眠や手間を大きく減らすか
3. 使う時期が今すぐか、数か月後か

### まず結論

| 品目 | 本当にいるか | 買う時期 | よいものの条件 |
|---|---|---|---|
| 鼻吸い器 | あるとよい。鼻がかめない0〜2歳では使用頻度が高い | 出産前は手動で可。電動は鼻風邪が増えてから | 洗いやすい、ノズルが柔らかい、吸引力を調整しやすい |
| ベビーピンセット | 必須ではない。乾いた鼻くそが見える時だけ | 産後に必要を感じてからでよい | 先端が丸い、短い、キャップ付き、洗いやすい |
| 哺乳瓶スチーム除菌・乾燥機 | 混合・完ミなら便利。完母予定なら後回し | 混合・完ミ予定なら産前検討。完母予定なら産後判断 | 乾燥機能、容量、置き場所、部品の洗いやすさ |
| おむつ専用ゴミ箱 | 防臭袋で足りる家庭も多い。臭い・動線で困るなら有用 | 産後に臭いが気になってから。離乳食開始後に再検討 | 密閉性、片手開閉、市販袋対応、丸洗いしやすい |
| 搾乳器 | 全員必須ではない。授乳状況で必要性が決まる | 産後判断が基本。早産・NICU・復職予定なら早め | 痛くない、サイズ調整、洗いやすい、部品購入可能 |
| 指挟み防止グッズ | 動き出したら安全用品として必要度が上がる | 寝返り〜ずりばい前後に危険箇所から | 外れにくい、賃貸でも使える、扉の開閉を妨げすぎない |

### 鼻吸い器

赤ちゃんは自分で鼻をかめません。  
鼻水が多いと、飲みにくい、寝にくい、機嫌が悪い、咳き込みやすい、という形で生活全体に影響します。

出産前に高額な電動タイプまで買う必要はありません。  
まずは手動タイプを1つ用意し、鼻風邪が多い、保育園入園予定、夜に鼻づまりで眠れないことが多い、という状況になったら電動を検討します。

| タイプ | 向く家庭 | 注意 |
|---|---|---|
| スポイト型 | 安い、短期使用 | 吸引力が弱め。洗いにくい形は避ける |
| 口吸いチューブ型 | 吸う力を調整しやすい | 親が風邪をもらいやすいと感じる家庭もある |
| 電動ハンディ型 | 外出・寝室で使いやすい | 据え置き型より吸引力は控えめなことがある |
| 電動据え置き型 | 鼻風邪が多い、保育園、兄弟がいる | 高額・場所を取る。部品洗浄が続くか確認 |

### 鼻吸い器の使い方の注意

- [ ] 奥まで入れない
- [ ] 強く吸いすぎない
- [ ] 嫌がる時に長時間押さえつけない
- [ ] 使用後は毎回洗う
- [ ] 発熱、哺乳不良、呼吸が苦しそうな時は小児科に相談する

鼻吸い器は「病気を治す道具」ではなく、鼻水で飲む・眠るが邪魔されるのを減らす補助具です。  
ゼーゼーする、肩で息をする、唇の色が悪い、飲めない、ぐったりしている時は、鼻吸い器で様子見しすぎず医療相談を優先します。

### ベビーピンセット

ベビーピンセットは、鼻の入口に見えている乾いた鼻くそを取るための小物です。  
鼻水を吸う道具ではありません。

必要度は低めです。  
綿棒や鼻吸い器で足りる家庭も多いため、産後に「見えているのに取れない」が何度かあってから買っても遅くありません。

選ぶなら、次の条件を見ます。

- [ ] 先端が丸い
- [ ] 先端が細すぎない
- [ ] キャップがある
- [ ] 水洗いしやすい
- [ ] 大人が片手で扱いやすい

鼻の奥へ入れる使い方はしません。  
動く赤ちゃんに使うため、深追いすると粘膜を傷つけます。「入口に見えていて、軽く取れるものだけ」と決めます。

### 哺乳瓶スチーム除菌・乾燥機

哺乳瓶や搾乳器部品は、洗浄が基本です。  
除菌は洗浄の代わりではありません。ミルクや母乳の残りがついたままスチームにかけても、清潔な運用にはなりません。

CDCは、搾乳器部品について、使用後は分解して洗い、十分に乾かすこと、特に生後2か月未満、早産、免疫が弱い赤ちゃんでは毎日の除菌が重要だと説明しています<sup>[15](#ref-cdc-pump-hygiene)</sup>。哺乳瓶でも考え方は近く、「洗う→除菌する→完全に乾かす→清潔に保管する」の順番が重要です。

| 授乳方針 | 除菌乾燥機の優先度 |
|---|---|
| 完母予定で哺乳瓶をほぼ使わない | 低い。産後判断でよい |
| 混合 | 中。毎日使うなら便利 |
| 完ミ | 高め。本数が多く、乾燥まで自動だと楽 |
| 双子・多胎 | 高め。洗浄・乾燥の手間が大きい |
| 早産・低出生体重・医師から衛生指導あり | 小児科・産科の指示を優先 |

### 選ぶポイント

- [ ] 乾燥機能がある
- [ ] よく使う哺乳瓶が入る
- [ ] 乳首、キャップ、搾乳器部品も入る
- [ ] 毎回の水捨て・水垢掃除が簡単
- [ ] 置き場所を取らない
- [ ] 部品を外して洗える
- [ ] スチーム対応不可の部品を入れない運用ができる

乾燥機能はかなり重要です。  
濡れたまま長時間置くと、清潔にしたつもりでも管理が難しくなります。乾燥機能がない場合は、清潔な場所でしっかり自然乾燥させます。

### おむつ専用ゴミ箱・防臭袋

新生児期の便は、離乳食後ほど強く臭わないことも多いです。  
最初は防臭袋だけで十分な家庭もあります。

一方で、次の家庭では専用ゴミ箱の満足度が上がります。

- [ ] 寝室やリビングでおむつ替えする
- [ ] ゴミ収集日まで室内保管が長い
- [ ] 夏場に臭いが気になる
- [ ] 集合住宅でベランダ保管しにくい
- [ ] 離乳食開始後の便臭が気になる
- [ ] 双子・兄弟でおむつ量が多い

### 選ぶポイント

| 見る点 | 理由 |
|---|---|
| 市販ゴミ袋が使えるか | 専用カートリッジはランニングコストが上がる |
| 片手で開くか | 抱っこ中・おむつ替え中に使いやすい |
| 密閉性 | 臭い漏れを減らす |
| 丸洗いしやすいか | 内側に臭いが残ることがある |
| 容量 | 大きすぎると長期保管になり臭う。小さすぎると交換が多い |
| 置き場所 | 寝床や調乳場所の近くに置きすぎない |

結論としては、「防臭袋を先に使い、臭いに困ったら専用ゴミ箱を買う」でよい家庭が多いです。

### 搾乳器

「授乳器」と呼ばれることがありますが、本書では、母乳を搾る道具を**搾乳器**と書きます。  
搾乳器は、母乳育児をする全員に必要な道具ではありません。

必要になりやすいのは、次の状況です。

- [ ] 赤ちゃんが直接飲みにくい
- [ ] 早産・NICU入院などで母子分離がある
- [ ] 胸が張ってつらい
- [ ] 母乳量を維持したい
- [ ] 家族が哺乳瓶で授乳する時間を作りたい
- [ ] 復職後も母乳を続けたい
- [ ] 外出・通院などで授乳間隔が空く

### 手動・電動の選び方

| タイプ | 向く使い方 |
|---|---|
| 手動 | たまに使う、安く始めたい、外出用 |
| 電動シングル | 1日数回まで、片胸ずつでよい |
| 電動ダブル | 頻回搾乳、復職、NICU、時短重視 |
| レンタル | 短期だが高性能が必要、産後に試したい |

### 選ぶポイント

- [ ] 吸引圧を細かく調整できる
- [ ] 痛くない
- [ ] フランジサイズを選べる
- [ ] 分解・洗浄・乾燥が簡単
- [ ] 交換部品が買える
- [ ] 母乳がチューブに入りにくい構造
- [ ] 音、重さ、充電方式が生活に合う

痛みが強い搾乳は、我慢して続けるものではありません。  
フランジサイズ、当て方、吸引圧、乳頭トラブル、乳腺炎リスクが関わるため、助産師や母乳外来に相談します。

### 扉・引き出しの指挟み防止グッズ

指挟み防止は、寝返り前の新生児期には優先度が低いです。  
ただし、寝返り、ずりばい、ハイハイ、つかまり立ちが始まると、扉、引き出し、収納、浴室、玄関、ベビーゲート周辺で急に必要になります。

| 時期 | 対策 |
|---|---|
| 0〜4か月 | すぐには不要。危険な扉を把握する |
| 5〜8か月 | 寝返り・ずりばい前に、よく使う扉から設置 |
| 8〜12か月 | 引き出し、収納、キッチン、洗面所を重点対策 |
| 1〜2歳 | 自分で開ける、閉める、登るため、ロックと見守りを強化 |

### 種類

| 種類 | 使う場所 |
|---|---|
| ドアストッパー | ドアが勢いよく閉まるのを防ぐ |
| 蝶番側カバー | ドアの吊元側に指が入るのを防ぐ |
| 引き出しロック | 収納や食器棚の開閉を防ぐ |
| 扉ロック | キッチン、洗面、薬・洗剤収納 |
| ソフトクッション | 軽い室内扉の閉まり切りを防ぐ |

指挟み対策は、商品を買って終わりではありません。  
子どもが外せる、粘着が弱い、開閉時に親が外しっぱなしにする、という失敗が起きます。設置後に大人が何度か開閉し、「毎日使っても続くか」を確認します。

### 買いすぎを防ぐルール

- [ ] 出産前に全部そろえない
- [ ] まず手動・低価格・小型で試す
- [ ] 高額品は、困りごとが3回以上起きてから検討する
- [ ] 衛生用品は「洗えるか」「乾くか」で判断する
- [ ] 安全用品は「子どもが外せないか」「親が使い続けられるか」で判断する

## 4-15. ぬいぐるみ・メリー

### 結論

ぬいぐるみは、最初のおもちゃとして買ってよいです。  
ただし、**遊び用**と**寝床用**を分けます。1歳までは、寝ている時の寝床には入れません。

メリーは、ベッド内に置く物ではなく、赤ちゃんが見上げる視覚刺激です。  
出産前に準備してもよいですが、赤ちゃんの手が届かない位置に固定し、寝返りや手伸ばしが始まったら外します。

### いつ買うか

| 品目 | 買う時期 | 判断 |
|---|---|---|
| ぬいぐるみ | 妊娠中に買ってよい | 思い出品・写真・起きている時の遊び用として可 |
| ベッドメリー | 妊娠中に買ってよい。設置は退院前に再確認 | 取り付け場所、落下、ひも、掃除を確認 |
| 床置きメリー/プレイジム | 産後〜3か月頃 | 寝床ではなく、起きている時の遊び場で使う |
| 寝床に入れる小さなぬいぐるみ | 1歳以降に検討 | 小さく軽く、ひも・ボタン・電池なし。必須ではない |

### ぬいぐるみの選び方

- [ ] 洗える
- [ ] 毛が抜けにくい
- [ ] 目・鼻が取れにくい
- [ ] ボタン、ビーズ、小さい飾りがない
- [ ] 長いリボン、ひも、ループがない
- [ ] 大きすぎない
- [ ] 重すぎない
- [ ] 対象年齢表示が新生児・乳児に合う

最初のぬいぐるみは、赤ちゃんが遊ぶものというより、親が見せて話しかける道具です。  
「くまさん来たね」「ふわふわだね」「おはようって言ってるね」と、親子の会話を増やす使い方にします。

### ぬいぐるみのNG運用

- [ ] 1歳未満の寝床に入れる
- [ ] 顔の横に置いたまま寝かせる
- [ ] 大きいぬいぐるみを枕のように使う
- [ ] 電池入り・音が鳴るものを寝床に入れる
- [ ] 長い毛やリボンを口に入れられる状態にする
- [ ] ベビーカーやチャイルドシートで顔にかかる位置に置く

AAPは、1歳未満の睡眠エリアから柔らかい物や緩い寝具を遠ざけることを推奨しています<sup>[16](#ref-aap-sleep)</sup>。ぬいぐるみは「買ってはいけない」ではなく、「寝ている時に寝床へ入れない」と理解します。

### メリーの選び方

| 見る点 | 理由 |
|---|---|
| 固定方式 | 落下防止 |
| ひも・コードの短さ | 首や手に絡むリスクを減らす |
| 手が届かない高さ | 引っ張り・部品外れ防止 |
| 音量調整 | 寝かしつけを妨げない |
| 掃除しやすさ | ほこり対策 |
| 電池交換のしやすさ | 外れた部品の管理 |
| 取り外しやすさ | 4〜5か月頃から見直しやすい |

### メリーの設置ルール

- [ ] ベッド内に置かない
- [ ] 赤ちゃんの手が届かない
- [ ] ひもやコードが垂れない
- [ ] 取り付け後に大人が軽く揺らして緩みを確認する
- [ ] ベッド柵に無理な力がかからない
- [ ] 寝返り、手伸ばし、四つばい、つかまり立ちが近づいたら外す
- [ ] 飾り用ガーランド、ライトコード、長いリボンはベッド周りに付けない

「妊娠初期からベッドにつけておく」こと自体は、赤ちゃんがまだ使わないため直ちに危険ではありません。  
ただし、出産までにほこりがたまる、取り付けが緩む、部屋の模様替えでコード位置が変わることがあります。退院前に一度外して掃除し、赤ちゃんを寝かせる前に固定と高さを再確認します。
