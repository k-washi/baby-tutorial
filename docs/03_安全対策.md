# 第3章　安全対策：睡眠・窒息・誤飲・事故予防

この章は最重要です。  
0〜2歳では、教育よりもまず事故予防です。

---

## 3-1. 安全睡眠：寝ている時の窒息対策

### 結論

1歳までは、赤ちゃんを**仰向け**に寝かせます。  
寝床は**硬く平ら**にします。  
寝床には**何も置きません**。

### 安全な寝床の条件

| 項目 | 正解 |
|---|---|
| 姿勢 | 仰向け |
| 寝床 | 硬く平らなマットレス・敷布団 |
| シーツ | ぴったりしたフィットシーツ |
| 掛け物 | 掛け布団ではなくスリーパー中心 |
| 親との距離 | 同室はよいが、同じ寝床は避ける |
| 室温 | 暑すぎを避ける。厚着させすぎない |
| 寝床に置く物 | 原則ゼロ |

### 寝床に置かないもの

- [ ] 枕
- [ ] 絶壁防止枕
- [ ] ドーナツ枕
- [ ] ぬいぐるみ
- [ ] 毛布
- [ ] 掛け布団
- [ ] 授乳クッション
- [ ] 寝返り防止クッション
- [ ] ベッドガード
- [ ] 傾斜クッション
- [ ] ソファ
- [ ] 大人用ベッドの柔らかいマットレス

### ぬいぐるみ・メリーはいつからよいか

「寝床に置かない」は、ぬいぐるみやおもちゃを買ってはいけないという意味ではありません。  
**寝ている時の顔周り・睡眠面・手の届く範囲に置かない**という意味です。

| 品目 | 起きている時 | 寝ている時 | 目安 |
|---|---|---|---|
| ぬいぐるみ | 大人が見ているなら0か月から遊び用に使える | 1歳までは寝床に入れない | 1歳以降に小さく軽いものを1つから検討 |
| 布おもちゃ | 大人が見ているなら可 | 1歳までは寝床に入れない | なめても洗えるものを選ぶ |
| メリー/モビール | 見る遊びとして使える | 触れない位置に固定されている場合のみ。落下・ひも・部品外れがあるなら不可 | 手を伸ばす、寝返り、ずりばい、四つばいが近づいたら外す |
| ベッド柵につけるおもちゃ | 起きていて大人が見ている短時間なら可 | 原則、寝る時は外す | ひも、クリップ、部品が手に届くものは避ける |

#### ぬいぐるみを「最初のおもちゃ」にする場合

ぬいぐるみは、親が見ている時間の遊び用ならよい選択肢です。  
写真を撮る、顔の近くで見せる、声を当てて話しかける、手で触らせる、腹ばい時間の少し前に置く、という使い方ができます。

選ぶ条件は次の通りです。

- [ ] 小さすぎる部品、ボタン、ビーズ、鈴がない
- [ ] 目や鼻が刺繍、または強く固定されている
- [ ] 長いリボン、ひも、ループがない
- [ ] 毛が抜けにくい
- [ ] 洗える
- [ ] 大きすぎず、重すぎない
- [ ] 対象年齢表示を確認した

赤ちゃんが眠くなったら、ぬいぐるみは寝床から出します。  
「抱いたまま寝たらかわいい」ではなく、「寝たら外す」が安全運用です。

1歳以降は、寝返り、座る、立つ、顔にかかった物を払う力が増えます。  
それでも、寝床に入れるなら小さく軽いものを1つからにし、大きいぬいぐるみ、長い毛、ひも付き、重いもの、電池入り、音が鳴るものは避けます。早産、呼吸器疾患、発達や運動に心配がある場合は、小児科に相談してからにします。

#### メリー/モビールをベッドにつける場合

メリーは、睡眠面に置く物ではなく、赤ちゃんが見上げる視覚刺激です。  
そのため、「寝床に何も置かない」とは少し扱いが違います。ただし、落下、ひも、部品外れ、赤ちゃんが引っ張ることがリスクになります。

妊娠中に取り付けておくこと自体は可能ですが、実際に赤ちゃんを寝かせる前に、次を確認します。

- [ ] 説明書どおりに固定した
- [ ] 赤ちゃんの手が届かない高さにある
- [ ] ひも、リボン、コードが垂れていない
- [ ] 小部品が外れない
- [ ] ベッド内に落ちる位置に重い部品がない
- [ ] 寝返り・手伸ばしが始まったら外す予定を決めた
- [ ] ほこりをためず、掃除できる

特に、手を伸ばして触る、寝返りする、四つばいに近づく、柵につかまる時期には外します。  
月齢だけでなく「届くか」「引っ張れるか」で判断します。一般には生後4〜5か月頃から見直しが必要です。

ベッド周りで避けたいのは、メリー以外にも、長い飾りひも、ガーランド、ベッド上のライトコード、カーテンやブラインドのコード、落ちそうな棚や額です。AAPは、乳児の睡眠エリアから柔らかい物や緩い寝具を遠ざけることに加え、垂れ下がるコード類も危険として避けるよう示しています<sup>[1](#ref-aap-sleep)</sup>。

### 根拠

AAPは、睡眠関連乳児死亡を減らすため、仰向け、硬く平らで傾斜のない寝床、柔らかい物や寝具を置かないことを推奨しています<sup>[1](#ref-aap-sleep)</sup>。日本の消費者庁も、就寝時の窒息事故予防として、1歳までは仰向け、硬めの敷布団、顔周りに物を置かないことを注意喚起しています<sup>[2](#ref-caa-sleep)</sup>。

### 論理的な理由

赤ちゃんは首や体幹の力が未熟で、顔が布や柔らかい物に埋もれても自力で逃げられないことがあります。  
「ふかふかで気持ちよさそう」は、大人の感覚では快適でも、赤ちゃんには窒息リスクになります。

### 心配性家庭への推奨セット

- [ ] ベビーベッドまたは安全な専用寝床
- [ ] 硬めのベビーマットレス
- [ ] フィットシーツ2〜3枚
- [ ] 防水シーツ2枚
- [ ] スリーパー2枚
- [ ] 室温湿度計
- [ ] 映像ベビーモニター
- [ ] 必要なら体動センサー

### 推奨セットの使い方

見慣れない育児用品は、「何のために使うか」と「どう置くか」を間違えないことが大切です。  
特に寝具は、赤ちゃんの顔周りにたるみ・段差・柔らかい布を作らない前提で選びます。

| 品目 | 役割 | 使い方・選び方 |
|---|---|---|
| ベビーベッドまたは安全な専用寝床 | 大人の布団・枕・掛け布団から赤ちゃんを分ける | 親と同室に置くのはよい。大人用ベッドやソファで一緒に寝かせる代わりに使う |
| 硬めのベビーマットレス | 顔が沈み込むリスクを下げる | ベッドの内寸に合うものを選ぶ。手で押して深く沈むもの、ふかふかしたもの、傾斜付きは避ける |
| フィットシーツ | マットレス表面を清潔に保ち、たるみを作らない | ゴムでマットレスにぴったり固定するシーツ。余った布を折り込むだけの大人用シーツは使わない |
| 防水シーツ | おしっこ・吐き戻しからマットレスを守る | 基本は「マットレス → 防水シーツ → フィットシーツ」の順。防水シーツを一番上に露出させない |
| スリーパー | 掛け布団の代わりに体を冷えにくくする | 首元・袖口が大きすぎず、顔にずり上がらないサイズを選ぶ。厚着させすぎない |
| 室温湿度計 | 暑すぎ・乾燥しすぎに気づく | 寝床の近く、赤ちゃんの手が届かない場所に置く。エアコンの風が直接当たる場所や窓際は避ける |
| 映像ベビーモニター | 別室から姿勢や泣きに気づく | カメラやコードは寝床の外に置く。ベッド柵にコードを垂らさない |
| 体動センサー | 親の不安を下げる補助 | 安全睡眠の代わりにはならない。誤作動がありうるので、使うなら説明書どおりに設置する |

寝具の重ね方は、次の形を基本にします。

```text
赤ちゃん
フィットシーツ
防水シーツ
硬めのベビーマットレス
ベビーベッド・専用寝床
```

防水シーツは「赤ちゃんに直接触れるシーツ」ではなく、マットレスを汚れから守るための内側の層です。  
上に出したままだと、素材によっては蒸れたり、ずれたり、顔周りにたるみができたりします。必ず上からフィットシーツで固定します。

フィットシーツと防水シーツは、吐き戻しやおむつ漏れで夜中に交換することがあります。  
最低限は各2枚、洗濯頻度を下げたい家庭はフィットシーツ3枚、防水シーツ2〜3枚あると運用しやすいです。

室温・湿度は、厳密な数字よりも「暑すぎない」「厚着させすぎない」ことを優先します。目安は次の通りです。

| 項目 | 目安 | 見るポイント |
|---|---|---|
| 室温 | 大人が薄着で快適な温度 | 赤ちゃんの背中や胸が汗ばんでいないか |
| 湿度 | 40〜60%程度 | 乾燥で咳・鼻づまりが強くないか、結露やカビが出ていないか |
| 服装 | 大人より1枚多いとは限らない | 手足の冷たさだけで判断せず、首の後ろ・背中・胸を見る |

夏は冷房を使ってよいですが、冷風を赤ちゃんに直接当てないようにします。  
冬は暖めすぎ、厚手スリーパー、帽子、掛け布団の重ねすぎに注意します。汗をかいている、首の後ろが熱い、顔が赤い時は暑すぎのサインです。

## 3-2. 呼吸観察装置・ベビーモニターの位置づけ

### 結論

見守り機器は、**親の安心を補助する道具**です。  
SIDSや窒息を予防する魔法の装置ではありません。

### 種類別の使い方

| 種類 | 使い道 | 注意点 |
|---|---|---|
| 映像モニター | 別室から様子を見る | 呼吸停止を予防するものではない |
| 音声モニター | 泣き声・物音に気づく | 映像がないので姿勢確認はできない |
| マット型体動センサー | 体動低下時にアラーム | 誤作動・過信に注意 |
| ウェアラブル型 | 心拍・酸素などを表示する機種もある | 医療機器でないものを診断・予防目的に使わない |
| 医療用モニター | 医師の指示で使用 | 一般家庭が自己判断で使うものではない |

### 根拠

FDAは、乳児のバイタルを監視してSIDSや睡眠関連死亡を予防すると主張する未承認デバイスに注意を促しています<sup>[3](#ref-fda-monitor)</sup>。AAPも家庭用心肺モニターをSIDS予防策として使うことは推奨していません<sup>[1](#ref-aap-sleep)</sup>。

### 使うなら守るルール

- [ ] 安全な寝床ルールを絶対に崩さない
- [ ] アラームが鳴った時の確認手順を夫婦で決める
- [ ] 誤作動で親が眠れなくなりすぎる場合は使い方を見直す
- [ ] 医療判断に使わない
- [ ] 体調不良時は機械より小児科相談を優先する

## 3-3. 食事中の窒息・誤嚥対策

この節は、主に**離乳食を始める生後5〜6か月頃以降**の話です。  
生後0〜4か月頃は、基本的に母乳・ミルクの時期なので、固形物による窒息対策ではなく、授乳姿勢、哺乳後の様子、吐き戻しへの対応が中心になります。

ただし、窒息リスクは離乳食開始直後だけの話ではありません。  
手づかみ食べが増える生後9か月頃から、歩きながら食べたがる1〜2歳頃まで、特に注意が必要です。

| 時期 | この節の関係 |
|---|---|
| 0〜4か月頃 | 固形物は原則なし。母乳・ミルク中心 |
| 5〜6か月頃 | 離乳食開始。なめらかなペースト中心で、姿勢と見守りが重要 |
| 7〜8か月頃 | つぶした食材・やわらかい小片が増える。丸い形・硬いものは避ける |
| 9〜11か月頃 | 手づかみ食べが始まりやすい。詰め込み、丸のみ、歩き食べに注意 |
| 1〜2歳頃 | 食べられるものが増えるが、ぶどう・ミニトマト・ナッツ・飴などは引き続き危険 |

### 危険な食べ物と対策

| 食べ物 | 対策 |
|---|---|
| ぶどう | 4分割。丸のまま出さない |
| ミニトマト | 4分割 |
| うずら卵 | 丸のまま避ける。小さく切る |
| ナッツ・豆 | 乳幼児期は避ける |
| 飴・グミ | 避ける |
| もち | 2歳頃までは基本避ける |
| はちみつ | 1歳未満は禁止。乳児ボツリヌス症予防<sup>[4](#ref-mhlw-honey)</sup> |
| パン | 小さく、飲み込みやすく。詰め込み注意 |
| りんご・にんじん | 月齢に応じて加熱・薄切り・すりおろし |

こども家庭庁や消費者庁は、ミニトマトやぶどうを4分割すること、食事中に走る・泣く・笑う・話すことを避けることを示しています<sup>[5](#ref-cfa-choking)</sup>。

### 食事中のルール

- [ ] 座って食べる
- [ ] 歩きながら食べない
- [ ] 車の中で一人で食べさせない
- [ ] 泣いている時に口へ入れない
- [ ] 口に詰め込ませない
- [ ] 食事中は大人が見ている
- [ ] 食材は月齢に合わせて切る・加熱する

## 3-4. 誤飲対策

### 特に危険なもの

| 危険物 | 理由 |
|---|---|
| ボタン電池 | 食道や胃腸に重い損傷を起こす可能性 |
| 磁石 | 複数飲むと腸を挟む可能性 |
| 水で膨らむボール | 体内で膨張する危険 |
| 薬 | 少量でも危険なものがある |
| たばこ・ニコチン製品 | 中毒リスク |
| 洗剤・漂白剤 | 化学損傷 |
| 小さいおもちゃ | 窒息・誤飲 |

### 家庭の対策チェック

- [ ] ボタン電池を使う製品は電池蓋を確認する
- [ ] リモコン・体温計・ライト類を床に置かない
- [ ] 磁石玩具は2歳までは原則避ける
- [ ] 薬は高い場所ではなくロック付き収納
- [ ] 洗剤は扉ロックのある場所へ
- [ ] 直径4cm未満の小物を床から消す
- [ ] 上の子のおもちゃを赤ちゃんエリアに置かない
- [ ] 誤飲疑い時に持参するため、商品の外装を捨てすぎない

## 3-5. 水・風呂・転落・やけど

### 水回り

赤ちゃんは少量の水でも危険です。  
沐浴中、風呂、洗面台、バケツ、トイレでは、**一瞬でも目を離さない**ことを原則にします。

- [ ] 風呂場に一人で入れない
- [ ] 浴槽の残り湯を放置しない
- [ ] 沐浴中にインターホンへ出ない
- [ ] 上の子に見守りを任せない
- [ ] バスマットやベビーバスを使う時も手を添える

### 転落

- [ ] ソファに寝かせない
- [ ] 大人用ベッドに一人で置かない
- [ ] おむつ替え台から離れない
- [ ] ベビーチェアのベルトを使う
- [ ] ハイチェアに立たせない
- [ ] ベビーカーのベルトを毎回つける

### やけど

- [ ] 抱っこしながら熱い飲み物を飲まない
- [ ] テーブルクロスを垂らさない
- [ ] 電気ケトルのコードを届かない位置へ
- [ ] ミルクの温度確認をする
- [ ] 風呂の温度を先に確認する

### 扉・引き出し・指挟み

寝返り前は優先度が低いですが、寝返り、ずりばい、ハイハイ、つかまり立ちが始まると、扉や引き出しの指挟みリスクが上がります。  
特に、子どもは「閉まる側」だけでなく、ドアの蝶番側にも指を入れます。

- [ ] よく使う室内扉にドアストッパーを付ける
- [ ] 蝶番側に指が入る扉はカバーを検討する
- [ ] キッチン、洗面、薬、洗剤の扉はロックする
- [ ] 引き出しは開け閉め遊びが始まる前にロックする
- [ ] 玄関、浴室、ベランダへつながる扉は大人が管理する
- [ ] 粘着式グッズは、子どもが外せないか定期的に確認する

家中すべてに最初から付ける必要はありません。  
まずは、赤ちゃんが過ごす部屋、キッチン、洗面、浴室、玄関につながる動線から優先します。詳しい購入判断は[第4章 4-14](04_購入品完全ガイド.md)にまとめています。

## 3-6. 揺さぶりを絶対に避ける

赤ちゃんが泣き止まない時でも、激しく揺さぶってはいけません。  
泣きが強い時は、親が一時的に安全な寝床に置いて離れ、深呼吸することも必要です。

### 泣き止まない時の手順

1. おむつ
2. 授乳・ミルク
3. 暑い/寒い
4. げっぷ
5. 抱っこ・ゆらゆら
6. 暗く静かな部屋
7. 安全な寝床に置いて親が数分休む
8. いつもと違う泣き方、発熱、嘔吐、ぐったりなら相談

こども家庭庁や小児科系資料は、乳児を激しく揺さぶらないこと、泣きへの対処を家族で共有することを示しています<sup>[6](#ref-cfa-shaken)</sup>。
