# 第6章　0〜2歳 月齢別ロードマップ

CDCの発達マイルストーンは、多くの子がその月齢までにできることを示す目安で、親子と専門職の会話を助ける目的があります<sup>[1](#ref-cdc-milestones)</sup>。  
発達には幅があります。1項目できないだけで焦る必要はありませんが、心配がある時は健診や小児科で相談します。

## 6-2. 0〜1か月：生き延びる時期

### 成長の意味

この時期は、教育よりも生命維持と安心が中心です。  
赤ちゃんは泣くことでしか要求を伝えられません。

### 1日の流れ

- 授乳/ミルク
- おむつ替え
- 寝る
- 泣く
- 沐浴
- 体重・尿・便の確認

### 親がやること

- [ ] 仰向けで寝かせる
- [ ] 顔周りに何も置かない
- [ ] 授乳・排泄を記録する
- [ ] 体温・機嫌を見る
- [ ] 保湿する
- [ ] 親も寝る
- [ ] 来客を制限する

### 教育的にやること

- 顔を見る
- 声をかける
- 抱っこする
- 歌う
- 目が合ったら返す

### 抱っこ・寝かせる・起きている時間のバランス

0〜1か月は、育てるというより「回復しながら生かす」時期です。  
赤ちゃんは長く起きて遊ぶ段階ではありません。基本は、授乳・ミルク、おむつ替え、少し抱っこ、寝る、のくり返しです。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 眠っている | 安全な寝床に仰向けで寝かせる | 抱っこのまま親が寝落ちしない |
| 泣いている | 授乳、おむつ、暑さ寒さ、抱っこで確認 | 泣きは要求のサイン。すぐ完璧に泣き止ませなくてよい |
| 起きていて落ち着いている | 顔を見て声をかける程度 | 遊び込むより安心を作る |
| 親が限界 | 安全な寝床に置いて数分離れる | 揺さぶらないための安全策 |

この時期は「寝かせすぎではないか」と心配しすぎなくて大丈夫です。  
起きている時間に少し顔を見る、声をかける、抱っこするだけで十分な関わりになります。

## 6-3. 1〜3か月：見る・聞く・反応する時期

### 成長の意味

顔を見る、音に反応する、少し笑う、声を出す。  
親子のやりとりが始まります。

### やること

- [ ] 起きている時に短い腹ばい遊び（タミータイム）
- [ ] 左右から話しかける
- [ ] 赤ちゃんの声をまねる
- [ ] 目を見て話す
- [ ] 短い絵本を読む
- [ ] 散歩を始める

### 抱っこ・寝かせる・床で遊ぶのバランス

1〜3か月の赤ちゃんは、まだ「起きて長く遊ぶ」時期ではありません。  
基本は、**寝る、飲む、少し起きる、また寝る**のくり返しです。抱っこは甘やかしではなく、安心、授乳、寝かしつけ、げっぷ、泣きへの対応として自然に必要になります。

ただし、起きている時間をすべて抱っこやバウンサーで過ごす必要はありません。  
機嫌がよい短い時間に、仰向けで床に寝かせて顔を見て話す、左右から声をかける、短い腹ばい遊びをする、という「体を自由に動かす時間」も少し入れます。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 眠そう・授乳後に落ち着いている | 安全な寝床に仰向けで寝かせる | 寝る力と親の休息を守る |
| 泣く・不安そう・げっぷが必要 | 抱っこする | この時期の抱っこは安心と調整の手段 |
| 起きていて機嫌がよい | 床で仰向け遊び、短い腹ばい遊び | 首・背中・手足を自由に動かす |
| 親が疲れている | 安全な寝床に置いて休む | 泣いていても、安全を確保して数分離れてよい |

時間で決めるなら、「起きている時間の中に、短い床遊びを何回か入れる」程度で十分です。  
抱っこを減らすことが目的ではありません。赤ちゃんが落ち着く抱っこと、赤ちゃんが自分で体を動かす床時間の両方を作ることが目的です。

### 腹ばい遊び（タミータイム）

タミータイムとは、赤ちゃんが**起きている時に、大人が見守りながら、短時間だけうつぶせ姿勢で遊ぶ時間**のことです。  
日本語では「腹ばい遊び」と考えるとわかりやすいです。

目的は、うつぶせで寝かせることではありません。  
首、背中、肩、腕の力を少しずつ使い、頭を上げる、左右を見る、手で床を押す、寝返りやハイハイにつながる体の使い方を経験するための遊びです。

### やり方

1. 赤ちゃんが起きていて、機嫌がよい時に行う
2. 硬めのマットや床の上に、薄いプレイマットを敷く
3. 赤ちゃんを短時間だけ腹ばいにする
4. 顔の近くにタオル、枕、ぬいぐるみ、柔らかい布を置かない
5. 大人が必ずそばで見て、苦しそうならすぐ仰向けに戻す

最初は数十秒〜1分でも十分です。  
慣れてきたら、1日に何回か、短く分けて行います。泣く、顔を横にできない、疲れて頭が下がる時は、無理に続けません。

### 寝る時との違い

ここが最重要です。

| 場面 | 姿勢 | 理由 |
|---|---|---|
| 寝る時 | 仰向け | 窒息や睡眠中の事故を避けるため |
| 起きて遊ぶ時 | 大人が見ている短時間の腹ばい | 首・背中・肩・腕を使う経験になるため |

つまり、**睡眠は仰向け、遊びは見守りながら短い腹ばい**です。  
赤ちゃんが腹ばいのまま眠りそうになったら、仰向けに戻します。

タミータイムに関する研究レビューでは、起きている時の腹ばい時間が、首すわりや寝返りなどの大きな運動発達、腹ばいでの移動、ハイハイ、後頭部の平らさの予防と関連することが報告されています<sup>[2](#ref-tummy)</sup>。  
ただし、これは「長時間うつぶせにすればよい」という意味ではありません。安全に見守れる短い時間を、毎日の遊びに少し入れる、という考え方です。

### 「首がすわる」とは何か

「首がすわる」とは、赤ちゃんが頭を完全に固定できるという意味ではありません。  
重い頭を、首、背中、胸、肩まわりの筋肉で支え、姿勢が変わっても頭がぐらぐらしにくくなる状態です。

生まれたばかりの赤ちゃんは、頭が体に対して大きく、首や体幹の筋力も未熟です。  
そのため、縦抱きにすると頭が後ろへ倒れたり、左右に揺れたりします。成長とともに、目で人や物を見る、抱っこで姿勢を保つ、床で手足を動かす、短い腹ばい遊びで頭を少し持ち上げる、といった経験を通じて、少しずつ頭を支える力が育ちます。

| 見るポイント | 意味 |
|---|---|
| 縦抱きで頭のぐらつきが減る | 首と体幹で頭を支え始めている |
| 腹ばいで一瞬でも頭を持ち上げる | 首・背中・肩を連動して使い始めている |
| 仰向けで左右に顔を向ける | 見たい方向へ頭を動かす力が出ている |
| 引き起こしで頭が大きく遅れにくい | 体幹と首の協調が進んでいる |

首すわりは、抱っこを楽にするためだけの発達ではありません。  
頭を支えられるようになると、視線を自分で向けやすくなり、手を伸ばす、寝返りの準備をする、周囲を観察する、といった次の発達につながります。

ただし、首すわりの時期には個人差があります。  
「何か月何日までに完成」と考えるより、健診で医師や保健師に確認しながら見ます。ぐったりしている、左右差が強い、まったく頭を持ち上げる様子がない、哺乳や反応も気になる時は、小児科や健診で相談します。

## 6-4. 4〜6か月：寝返り・手を伸ばす・離乳食準備

### 成長の意味

「見る」から「触る」へ移ります。  
手を伸ばし、口に入れ、体をひねり、寝返りの準備をします。

### やること

- [ ] 床で遊ぶ時間を増やす
- [ ] 手が届く少し先におもちゃを置く
- [ ] 鏡や音の出るおもちゃで腹ばいを促す
- [ ] 離乳食の開始サインを見る
- [ ] ベビーチェアを検討する
- [ ] 予防接種スケジュールを確認する

### 抱っこ・寝かせる・床遊びのバランス

4〜6か月は、首や体幹がしっかりし始め、見るだけでなく「手を伸ばす」「体をひねる」「口で確かめる」が増える時期です。  
この頃から、起きていて機嫌がよい時間は、抱っこだけでなく床で自由に動ける時間を増やします。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 眠そう | 安全な寝床で仰向けに寝かせる | 寝返りが始まる前後でも寝床には物を置かない |
| 甘えたい・疲れた | 抱っこする | 抱っこは休憩と安心の場所 |
| 機嫌がよい | 床で仰向け、横向き、腹ばい遊び | 手を伸ばす、体をひねる経験を作る |
| 家事中 | 短時間なら安全な床エリアやベビーベッド | バウンサーや椅子に長時間固定しない |

この時期は「抱っこで見る世界」と「床で自分の体を使う世界」の両方が必要です。  
寝返りの準備は、親が練習として転がすより、赤ちゃんが床で手足を動かし、横を向き、興味のある物へ手を伸ばす中で進みます。

## 6-5. 7〜9か月：ずりばい・おすわり・探索

### 成長の意味

赤ちゃんが自分で移動し始めます。  
これは、運動だけでなく、空間認知、問題解決、自己効力感に関わる重要な変化です。

### やること

- [ ] 床の安全エリアを作る
- [ ] ベビーゲートを設置する
- [ ] コンセント・コードを対策する
- [ ] 小物を床から消す
- [ ] ハイハイしやすい服にする
- [ ] 離乳食を進める
- [ ] 手づかみ食べの準備をする

### 抱っこ・寝かせる・探索のバランス

7〜9か月は、抱っこで運ぶ時期から、赤ちゃんが自分で世界に近づき始める時期へ移ります。  
ずりばい、方向転換、おすわり、手を伸ばす動きが増えるため、起きている時間は「安全な床で探索」が中心になります。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 眠い | 安全な寝床で寝かせる | 寝床と遊び場を分ける |
| 不安・後追い・疲れた | 抱っこで落ち着かせる | 抱っこは探索に戻るための基地 |
| 機嫌がよい | 安全エリアでずりばい・おすわり・探索 | 小物、コード、段差を先に消す |
| 離乳食前後 | 椅子に座る時間は目的を限定 | 食事以外で長く固定しない |

この時期の床時間は、単なる運動ではありません。  
「あのおもちゃまで行きたい」「届かないから向きを変える」「親の顔を見て安全確認する」という経験が、運動、認知、親子のやりとりを同時に育てます。

## 6-6. 10〜12か月：ハイハイ・つかまり立ち・指差し

### ハイハイの意味

ハイハイは「歩く前のつなぎ」ではありません。

| 視点 | 意味 |
|---|---|
| 運動 | 肩、腕、手首、体幹、股関節を使う |
| 感覚 | 手のひら、膝、足裏で床を感じる |
| 空間認知 | 距離、段差、障害物を経験する |
| 視覚 | 近くと遠くを見比べる |
| 認知 | 目的物までどう行くか試行錯誤する |
| 社会性 | 親の顔を見て安全確認する |
| 自主性 | 自分で世界に近づく |

運動発達研究では、自力で移動できるようになることが、探索、空間判断、危険判断、親子の参照行動に影響することが示唆されています<sup>[3](#ref-locomotion)</sup>。

### ただし焦らない

ハイハイしない子、短期間しかしない子、すぐ立つ子もいます。  
問題視すべきなのは、強い左右差、発達の後退、極端な反り、ぐにゃぐにゃ、移動意欲が乏しいなどです。

### 抱っこ・寝かせる・移動遊びのバランス

10〜12か月は、ハイハイ、つかまり立ち、伝い歩き、指差しなどが重なり、事故リスクも一気に増えます。  
起きている時間は、親がずっと抱っこして守るより、危険を取り除いた環境で動けるようにする方が現実的です。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 眠い | 安全な寝床で寝かせる | 立ち上がる場合はベッド周りの安全も確認 |
| 甘えたい・眠い・不安 | 抱っこする | 情緒の補給。動く時間と対立しない |
| 機嫌がよい | ハイハイ、つかまり立ち、伝い歩きの安全エリア | 転ばせないより、危険な角・段差・小物を消す |
| 外出中 | 抱っこ紐・ベビーカーを使う | 帰宅後に床で動く時間を作る |

この時期は「歩かせる練習」より、ハイハイやつかまり立ちを安全に試せる環境が大切です。  
手押し歩行器や大人が両手を引いて長く歩かせることを急ぐより、床で自分の体重を支え、転び方を少しずつ学ぶ方が自然です。

## 6-7. 12〜18か月：歩く・まねる・言葉の前夜

### 成長の意味

歩行、指差し、まね、簡単な言葉が増えます。  
知育は「教え込む」より、生活に言葉をつけることが重要です。

### やること

- [ ] 外遊びを増やす
- [ ] 指差しに毎回反応する
- [ ] 生活を実況中継する
- [ ] 簡単なお手伝いを頼む
- [ ] 絵本を毎日読む
- [ ] 靴を実寸で選ぶ

### 抱っこ・休ませる・歩く時間のバランス

12〜18か月は、歩き始めても体力や判断力はまだ未熟です。  
「自分で歩きたい」と「疲れて抱っこしてほしい」が短時間で入れ替わります。どちらも自然です。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 眠い・疲れた | 昼寝や休憩を優先 | 機嫌の悪さは疲労のことが多い |
| 外で歩きたがる | 安全な場所で歩かせる | 道路、駐車場、階段では手をつなぐ |
| 抱っこを求める | 必要に応じて抱っこする | 甘えだけでなく、疲れ・不安・眠気の調整 |
| 家の中 | 低い段差、押す、運ぶ、入れる遊び | 歩くだけでなく全身を使う |

この時期の抱っこは「赤ちゃんに戻る」ことではありません。  
歩いて探索し、疲れたら抱っこで回復し、また歩く、という往復が自立の土台になります。

## 6-8. 18〜24か月：イヤイヤ・二語文・ごっこ遊び

### 成長の意味

イヤイヤは、反抗ではなく自己主張の発達です。  
ただし、脳のブレーキは未熟なので、言葉で落ち着くのは難しいことが多いです。

### 対応の基本

```text
気持ちを代弁する
　↓
ルールを短く言う
　↓
選択肢を出す
　↓
できた行動を具体的に褒める
```

### 抱っこ・休ませる・動く時間のバランス

18〜24か月は、体はよく動きますが、感情のブレーキはまだ未熟です。  
眠い、空腹、暑い、退屈、刺激が多いだけで、イヤイヤや癇癪が強くなることがあります。抱っこは、この時期も「甘やかし」ではなく、切り替えや安全確保の手段になります。

| 場面 | 目安 | 考え方 |
|---|---|---|
| 癇癪・危険行動 | まず安全確保。必要なら抱っこで離す | 説得より安全が先 |
| 眠い・空腹 | 休憩、昼寝、補食を優先 | しつけではなく体調調整の問題か見る |
| 外遊び | 走る、登る、投げる、運ぶ | 体を使うと睡眠と機嫌が整いやすい |
| 家の中 | 静かな遊びと動く遊びを分ける | 刺激が強すぎる時は環境を下げる |

この時期は「長く寝かせておく」より、午睡と夜の睡眠を守りつつ、起きている時間にしっかり体を使う方が生活が安定しやすくなります。  
ただし、疲れすぎると癇癪が増えるため、外遊び、食事、昼寝の順番を家庭のリズムとして固定します。

### 例

| 場面 | 対応 |
|---|---|
| まだ遊びたい | 「まだ遊びたかったね。あと1回でおしまい」 |
| 叩く | 「叩かない。痛い。こっちを叩こう」 |
| 食べない | 「おしまいにする？一口だけにする？」 |
| 着替えない | 「赤い服と青い服、どっち？」 |
| 癇癪 | 安全確保、短い言葉、待つ |

AAPは、体罰や言葉による虐待を避け、肯定的なしつけを推奨しています<sup>[4](#ref-aap-discipline)</sup>。
