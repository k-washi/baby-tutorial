# 第7章　授乳・ミルク・離乳食・アレルギー

## 7-1. 授乳・ミルクの基本

母乳でもミルクでも、赤ちゃんが育ち、親が続けられることが大切です。  
授乳方法は、産院・小児科・助産師と相談しながら決めます。

授乳で最初に大切なのは、「母乳かミルクか」という思想ではなく、赤ちゃんに水分とエネルギーが入り、体重が増え、親が倒れないことです。  
母乳は免疫成分や親子の接触などの利点があります。一方で、ミルクは量が見えやすく、他の家族が授乳を交代でき、親の睡眠を守りやすい利点があります。混合栄養は、その中間の現実的な選択肢です。

### 用語補足

| 用語 | 意味 |
|---|---|
| 完母 | 完全母乳。原則として母乳だけで育てること |
| 完ミ | 完全ミルク。粉ミルク・液体ミルクなどで育てること |
| 混合 | 母乳とミルクを併用すること |
| 直母 | 赤ちゃんが乳房から直接飲むこと |
| 搾乳 | 母乳を搾って哺乳瓶などで飲ませること |
| 哺乳 | 母乳やミルクを飲むこと |

### 見るポイント

- [ ] 体重が増えている
- [ ] 尿が出ている
- [ ] 便の様子が極端でない
- [ ] 哺乳後に苦しそうでない
- [ ] 親が限界を超えていない

授乳量の評価は、1回ごとの飲んだ量だけでは判断しません。  
母乳では量が見えないため、尿、便、体重、機嫌、哺乳の力、眠り方を合わせて見ます。ミルクでは量が見えますが、缶の表示は目安であり、毎回ぴったり飲む必要はありません。

| 観察 | 見方 |
|---|---|
| 尿 | 回数が極端に少ない、色が濃い、元気がない時は相談 |
| 便 | 回数だけでなく、機嫌、腹部の張り、血便の有無を見る |
| 体重 | 健診や小児科で増え方を確認。家庭の体重計だけで判断しすぎない |
| 哺乳後 | 毎回むせる、苦しそう、チアノーゼ、ぐったりは医療相談 |
| 親の状態 | 痛み、睡眠不足、涙が止まらない、授乳が恐怖なら支援が必要 |

### Tips

- 完母を目指しても、ミルクを使うことは失敗ではありません。
- 混合は夜間や父親の参加に役立つことがあります。
- ミルク量で迷う時は、缶の表示だけでなく小児科や健診で相談します。
- 授乳後に少し吐き戻すだけならよくありますが、噴水のように吐く、体重が増えない、ぐったりする時は相談します。
- 授乳姿勢で親の肩・腰・手首を壊しやすいので、クッションや椅子の高さを調整します。
- 夜間授乳は「誰が起きるか」まで決めます。混合や搾乳を使う場合は、家族が1回代われるだけでも親の回復が変わります。

### 授乳で相談した方がよいサイン

- [ ] 生後早期に尿が極端に少ない
- [ ] 体重増加を健診や小児科で指摘された
- [ ] 哺乳中に毎回むせる、顔色が悪い
- [ ] 飲む力が弱く、すぐ疲れる
- [ ] 発熱、ぐったり、反応が弱い
- [ ] 乳房の強い痛み、発赤、発熱がある
- [ ] 親が眠れず、授乳が怖い、涙が止まらない

ここは根性で乗り切る領域ではありません。  
小児科、産院、助産師、自治体の産後ケアに早めにつなぎます。

## 7-2. 離乳食開始

厚生労働省の「授乳・離乳の支援ガイド」は、離乳開始をおおむね生後5〜6か月頃としています<sup>[1](#ref-mhlw-weaning)</sup>。

離乳食は、母乳やミルクを急にやめることではありません。  
母乳・ミルクだけでは少しずつ足りなくなる栄養を、食べ物から取る練習へ移行する過程です。最初の目的は「栄養を完璧に取る」より、口に入る、舌で動かす、飲み込む、食卓に慣れることです。

### 開始サイン

- [ ] 首がすわっている
- [ ] 支えると座れる
- [ ] 食べ物に興味を示す
- [ ] スプーンを口に入れても強く押し出さない
- [ ] 体調がよい
- [ ] 親に余裕がある日

開始サインがそろっていないのに、月齢だけで急ぐ必要はありません。  
一方で、アレルギーが心配だからという理由だけで、離乳食全体や特定食品を長く遅らせるのも基本方針ではありません。迷う場合は、乳児健診や小児科で「開始してよい状態か」を確認します。

### 最初に用意するもの

- [ ] ベビーチェア
- [ ] 離乳食スプーン
- [ ] 小皿
- [ ] お食事エプロン
- [ ] 冷凍トレー
- [ ] ハンドブレンダーまたはすり鉢
- [ ] 麦茶用スプーンまたはコップ

### 最初の進め方

最初は、赤ちゃん用スプーン1さじからで十分です。  
量を増やすより、「体調がよい日」「平日午前〜昼」「1種類ずつ」「食後の変化を見られる日」を優先します。

| 段階 | 目的 | 例 |
|---|---|---|
| 最初 | スプーンと飲み込みに慣れる | なめらかな10倍がゆを少量 |
| 慣れてきたら | 野菜や豆腐などを増やす | かぼちゃ、にんじん、豆腐など |
| たんぱく質 | 鉄やたんぱく質を少しずつ | 白身魚、豆腐、卵黄などを慎重に |
| 水分 | 食事中の練習 | スプーンやコップで少量 |

### 用語補足

| 用語 | 意味 |
|---|---|
| 離乳初期 | おおむね5〜6か月頃。なめらかにすりつぶした状態から始める時期 |
| 離乳中期 | おおむね7〜8か月頃。舌でつぶせる固さに進む時期 |
| 離乳後期 | おおむね9〜11か月頃。歯ぐきでつぶせる固さ、手づかみが増える時期 |
| 離乳完了期 | おおむね12〜18か月頃。食事から取る栄養が増える時期 |
| ペースト | 粒が少なく、なめらかにすりつぶした状態 |
| みじん切り | 小さく刻むこと。ただし初期から急に使うものではない |

### 親が迷いやすいTips

- 食べない日はあります。1回食べないだけで栄養不足にはなりません。
- 眠い時間、空腹すぎる時間、授乳直後すぎる時間は食べにくいことがあります。
- 口から出すのは「嫌い」ではなく、舌や口の動かし方が未熟なだけのことがあります。
- 味付けより、形状と姿勢の方が大事です。
- 口に入れたまま泣く、笑う、動くと窒息リスクが上がるので、座って落ち着いて食べます。
- はちみつは1歳未満に与えません。乳児ボツリヌス症予防のためです<sup>[4](#ref-mhlw-honey)</sup>。
- 離乳食アプリやSNSの量は目安です。赤ちゃんの体調、便、体重、機嫌を見て調整します。

### 受診・相談した方がよい時

- [ ] 食べると毎回強くむせる
- [ ] 食後にじんましん、嘔吐、咳、顔色不良が出る
- [ ] 体重増加や貧血を指摘された
- [ ] 湿疹が強く、卵などの開始が不安
- [ ] 便秘や下痢が長引く
- [ ] 親が怖くて離乳食を進められない

## 7-3. 食物アレルギー：遅らせればよいわけではない

### 結論

アレルギーが心配だからといって、卵・小麦・乳などを一律に遅らせる考え方は現在の主流ではありません。  
ただし、湿疹が強い、アトピー性皮膚炎がある、既に反応が出た、家族歴が強い場合は、小児科やアレルギー専門医に相談します。

日本の食物アレルギー関連ガイドでは、食物アレルギーの発症予防を目的に離乳食開始や特定食品の開始を遅らせることを一般的には推奨していません<sup>[2](#ref-allergy-guideline)</sup>。LEAP研究では、高リスク児における早期ピーナッツ摂取がピーナッツアレルギー発症を大きく減らしたことが示されました<sup>[3](#ref-leap)</sup>。

### まず知っておく考え方

食物アレルギーは、「食べたことがない食品を避け続ければ防げる」という単純な仕組みではありません。  
皮膚の湿疹やバリア機能の低下、遺伝的ななりやすさ、食べる時期、食べる量、腸や免疫の発達など複数の要因が関わります。だからこそ、湿疹が強い子では、食品を遅らせる前に皮膚の治療を相談することが重要です。

「早く食べさせれば何でも安全」という意味でもありません。  
大切なのは、離乳食を始められる状態になったら、体調のよい日に、少量から、1種類ずつ、小児科に相談できる時間帯に試すことです。

### 症状の見方

| 症状 | 家庭での扱い |
|---|---|
| 口の周りが少し赤い | 食材の刺激やよだれかぶれもある。写真を撮り、広がりを見る |
| じんましんが広がる | 食物アレルギーの可能性。小児科へ相談 |
| 嘔吐をくり返す | 相談対象。ぐったりや顔色不良があれば急ぐ |
| 咳、ゼーゼー、呼吸が苦しそう | 緊急性が高い。救急相談・医療機関へ |
| ぐったり、意識がぼんやり | 緊急対応 |
| 唇やまぶたが腫れる | ほかの症状を伴う場合は急ぐ |

初めての食品を食べた後に症状が出た時は、食べたもの、量、時間、症状、写真を記録します。  
自己判断で「少しだけなら大丈夫」と再挑戦せず、小児科で相談します。

### 実践ルール

- [ ] 初めての食材は平日午前〜昼に少量
- [ ] 体調がよい日にする
- [ ] 1種類ずつ試す
- [ ] 湿疹が強い場合は先に皮膚治療を相談
- [ ] 反応が出た食材は自己判断で再挑戦しない
- [ ] 呼吸症状、ぐったり、全身じんましんは緊急対応

### 卵・小麦・乳を始める時の考え方

卵、小麦、乳は、アレルギーが心配されやすい食品です。  
ただし、家庭ごとの湿疹の強さ、既往歴、家族歴、地域の小児科方針で進め方が変わります。本書では細かい分量や日数を固定せず、原則だけ示します。

| 食品 | 最初の考え方 |
|---|---|
| 卵 | しっかり加熱したものを少量から。湿疹が強い場合は先に相談 |
| 小麦 | うどんやパンがゆなど、月齢に合う形状で少量から |
| 乳 | ヨーグルトなどを使う時も少量から。ミルクアレルギー疑いがある場合は相談 |
| ピーナッツ・ナッツ | 窒息リスクがあるため粒のままは不可。導入は小児科に相談 |

アレルギー対策と窒息対策は別です。  
食べさせてよい時期でも、丸いもの、硬いもの、粒のナッツ、飴、ぶどうやミニトマトの丸ごとは危険です。こども家庭庁は、ぶどうやミニトマトを4分割するなど、食品による窒息事故を防ぐ工夫を示しています<sup>[5](#ref-cfa-choking)</sup>。

## 7-4. 手づかみ食べ

手づかみ食べは汚れますが、重要です。

### 意味

| 視点 | 意味 |
|---|---|
| 感覚 | 温度、硬さ、粘り、形を学ぶ |
| 手指 | つまむ、握る、離す |
| 口 | 一口量を学ぶ |
| 自主性 | 自分で食べる意欲 |
| 認知 | 原因と結果を学ぶ |

### 注意

- [ ] 丸い食材をそのまま出さない
- [ ] 固すぎる食材を避ける
- [ ] 口いっぱいに入れない
- [ ] 必ず座って食べる
- [ ] 親が見ている

### 汚れることへの考え方

手づかみ食べは、親にとっては掃除が増えるイベントです。  
しかし赤ちゃんにとっては、手で触って、硬さを知り、握る力を調整し、自分で口へ運び、一口量を失敗しながら学ぶ練習です。

汚れをゼロにしようとすると、親が疲れます。  
床に新聞紙や洗えるマットを敷く、袖付きエプロンを使う、食後にまとめて拭く、食べる場所を固定するなど、片付けの仕組みで対応します。

### 手づかみ向き・不向きの考え方

| 向きやすい | 注意が必要 |
|---|---|
| やわらかく煮た野菜スティック | 生のにんじん、りんごの大きな塊 |
| 小さく切ったやわらかいパン | 口の中で団子になりやすいパンの詰め込み |
| やわらかいおやき | 硬いせんべい、ナッツ、豆 |
| ほぐした魚や豆腐ハンバーグ | 骨が残った魚、弾力の強い食品 |

「手づかみ食べをしないと発達が遅れる」と決めつける必要はありません。  
感触が苦手な子、手が汚れるのを嫌がる子もいます。スプーンで食べる日、親が補助する日、自分で触る日を混ぜながら進めます。
