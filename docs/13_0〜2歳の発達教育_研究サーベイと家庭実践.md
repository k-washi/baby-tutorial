# 第13章　発達教育 研究編：サーベイと家庭実践

## 13-1. この章の結論

0〜2歳の最高の教育は、文字、数字、英語、フラッシュカードを早く教えることではありません。  
安全な環境の中で、赤ちゃんがよく寝て、よく動き、親とやりとりし、言葉を浴び、遊びを通して探索することです。

研究とガイドラインを家庭の実践に落とすと、優先順位は次の通りです。

1. 安全な睡眠、栄養、予防接種、皮膚、事故予防を整える
2. 赤ちゃんの声、視線、表情、指差しに返す
3. 一方的に聞かせるより、会話ターンを増やす
4. 0か月から絵本、歌、語りかけを生活に入れる
5. 起きている時は、床で自由に動く時間を作る
6. 高額教材より、単純なおもちゃと親子の遊びを優先する
7. スクリーンを、親子のやりとり、睡眠、運動、食事の代わりにしない
8. 親の睡眠とメンタルヘルスを守る
9. 叩く、怒鳴る、脅す、恥をかかせるしつけを避ける
10. 発達の心配は、家庭で抱え込まず、健診、小児科、自治体相談につなぐ

WHOなどが整理する乳幼児期の発達支援では、健康、栄養、安全、応答的養育、早期学習機会が相互に支えるものとされています<sup>[1](#ref-nurturing-care-framework)</sup><sup>[2](#ref-who-ecd-guideline)</sup>。日本の保育所保育指針解説でも、0〜2歳は心身の発達の基盤が作られる時期として、身近な人との関わり、生活、遊び、主体的な環境への働きかけが重視されます<sup>[3](#ref-hoiku-guideline)</sup>。

この章は「何を買うか」ではなく、「家庭の中で何を繰り返すか」を扱います。

## 13-2. 用語補足

| 用語 | 意味 |
|---|---|
| 発達教育 | 知識を教え込むことではなく、脳、身体、言葉、情緒、社会性が育つ環境を作ること |
| 応答的養育 | 赤ちゃんのサインに気づき、温かく、タイミングよく、わかりやすく返すこと |
| Serve and Return | 赤ちゃんの声、視線、指差しなどに大人が返す往復のやりとり |
| 会話ターン | 赤ちゃんが声や表情を出し、大人が返し、また赤ちゃんが返す一往復 |
| Parentese | 少し高め、ゆっくり、抑揚があり、短く、赤ちゃんの反応を待つ話し方 |
| 共同注意 | 赤ちゃんと大人が同じものを見て、注意を共有すること |
| 探索 | 見る、触る、なめる、動く、落とす、入れるなど、自分で試す行動 |
| 実行機能 | 待つ、切り替える、注意を向ける、衝動を少し止める力 |
| 象徴遊び | 積み木を電話に見立てる、ままごとをするなど、何かを別のものに見立てる遊び |
| Video deficit | 乳幼児が画面上の情報より、対面の人から学びやすい傾向 |
| 背景テレビ | 子どもが直接見ていなくても、同じ部屋でテレビや動画が流れている状態 |

## 13-3. エビデンス強度別：家庭で優先すること

この表の「強い」は、複数の研究、介入研究、専門機関のガイドラインが同じ方向を示すものです。  
「中程度」は有望だが、研究数、対象年齢、効果の大きさに限界があるものです。  
「限定的」は、悪いわけではないが、0〜2歳で発達効果を強く期待する根拠がまだ弱いものです。

| 優先度 | 家庭でやること | 主に育つ力 | 根拠の強さ | 家庭での結論 |
|---:|---|---|---|---|
| 1 | 応答的養育、Serve and Return | 愛着、言語、認知、社会性 | 強い | 声、視線、指差し、泣きに返す |
| 2 | 親子の会話ターン | 言語、共同注意、脳の言語ネットワーク | 強い | 一方的に聞かせるより、往復を増やす |
| 3 | 読み聞かせ | 語彙、共同注意、親子関係 | 強い | 0か月から短く毎日でよい |
| 4 | 床遊び、タミータイム、自由運動 | 粗大運動、探索、姿勢制御 | 強い | 起きている時に安全な床を使う |
| 5 | 単純なおもちゃと親子遊び | 問題解決、手指、想像力 | 中〜強 | 少数を出して、遊びを広げる |
| 6 | スクリーンを最小限にする | 言語、睡眠、身体活動、親子相互作用 | 中〜強 | 18か月未満は原則避ける |
| 7 | 歌、音楽、リズム遊び | 情緒、社会性、前言語コミュニケーション | 中 | 流すだけより親子で歌う |
| 8 | 親のメンタルヘルス支援 | 応答性、愛着、言語環境 | 中〜強 | 親を助けることが発達支援 |
| 9 | ポジティブなしつけ | 自己制御、情緒、親子関係 | 強い | 体罰、脅し、恥を避ける |
| 10 | バイリンガル環境 | 言語、文化的基盤 | 中 | 人とのやりとりなら問題ない |
| 11 | 高額知育教材 | 商品により異なる | 限定的 | 0〜2歳では優先度を下げる |
| 12 | 動画教材の流しっぱなし | 受動視聴 | 限定的〜注意 | 教育目的でも長時間は避ける |

## 13-4. 発達教育の基本モデル

0〜2歳の発達は、上に知識を乗せる前に、下の土台を安定させる必要があります。

```text
        言葉・知育・社会性
      ─────────────────
        遊び・探索・まね・共同注意
      ─────────────────
        寝返り・腹ばい・ハイハイ・歩行
      ─────────────────
        睡眠・栄養・皮膚・情緒の安定
      ─────────────────
        安全な環境・親の応答・愛着
```

赤ちゃんは、座って授業を受けることで発達するのではありません。  
安心できる大人が近くにいて、自分で見て、触って、動いて、試して、失敗して、またやることで発達します。

### 研究の読み方

育児情報では「この教材で賢くなる」「この動画で英語脳」など、強い言い方が出ます。  
研究を読む時は、次の4つを分けます。

| 種類 | わかること | 限界 |
|---|---|---|
| 観察研究 | ある行動と発達が関連するか | 因果とは限らない |
| 介入研究 | ある関わりを増やすと変化が出るか | 対象や期間に限界がある |
| メタ分析 | 複数研究を統合した全体傾向 | 研究の質がばらつく |
| ガイドライン | 専門家が研究と実践を踏まえて推奨 | 国、時期、対象で更新される |

家庭で大事なのは、「何か1つで発達が決まる」と考えないことです。  
睡眠、栄養、親の応答、遊び、安全、家庭ストレスは絡み合っています。

## 13-5. 応答的養育：最重要の発達教育

### 何をすることか

応答的養育とは、赤ちゃんの小さなサインに気づき、タイミングよく、温かく、わかりやすく返すことです。

```text
赤ちゃんが見る・声を出す・手を伸ばす・指差す
        ↓
親が気づく
        ↓
「見つけたね」「ワンワンだね」と返す
        ↓
赤ちゃんがもう一度反応する
        ↓
やりとりが続く
```

ハーバード大学Center on the Developing Childは、このような相互作用をServe and Returnとして説明しています<sup>[4](#ref-harvard-serve)</sup>。WHOの早期発達ガイドラインでも、0〜3歳では応答的養育と早期学習活動が重要な支援として位置づけられています<sup>[2](#ref-who-ecd-guideline)</sup>。

### なぜ効くのか

| 赤ちゃんの経験 | 育つ力 |
|---|---|
| 泣いたら来てくれる | 安心、安全感、愛着 |
| 声を出したら返してくれる | 会話の順番、音声模倣 |
| 指差したら名前を言ってくれる | 語彙、共同注意 |
| できたことを見てもらえる | 意欲、探索心 |
| 困った時に助けてもらえる | 情緒調整の土台 |

複数国の育児介入研究を統合したPLOS Medicineのメタ分析では、ペアレンティング介入が子どもの認知、言語、運動、社会情緒発達などに良い方向の効果を示し、応答的養育を含む介入が養育行動にも影響することが報告されています<sup>[5](#ref-plos-parenting-2021)</sup>。

### 限界

応答的養育は、親が24時間完璧に反応することではありません。  
赤ちゃんが泣くたびに即座に正解を出す必要もありません。

家庭での目標は、次の4つです。

1. 気づく
2. 返す
3. 待つ
4. もう一度返す

### 実践

| 場面 | やること |
|---|---|
| 目が合った | 微笑んで「見てるね」と返す |
| 声を出した | 同じような声で返す |
| 手を伸ばした | すぐ渡しすぎず、少し待ってから助ける |
| 泣いた | 抱く、声をかける、原因を探す |
| 指差した | 「電車だね。走ってるね」と言葉を足す |
| できた | 大げさに評価するより「できたね」と共有する |

## 13-6. 言葉の発達：量より「やりとり」

### 会話ターンが重要

乳児への語りかけは重要ですが、単語を大量に浴びせればよいわけではありません。  
近年の研究では、赤ちゃんの反応に大人が返し、また赤ちゃんが返す「会話ターン」が重視されます。

PNASに掲載された親へのコーチング介入研究では、親が乳児に向けた話し方を変え、parenteseと会話ターンを増やすことで、乳児の言語発達が促進されました<sup>[6](#ref-pnas-lang)</sup>。また、幼児期の研究では、成人と子どもの会話ターンが言語関連脳ネットワークの発達と関連することも報告されています<sup>[7](#ref-jns-lang)</sup>。

### Parenteseは「赤ちゃん言葉」だけではない

| 特徴 | 例 |
|---|---|
| 少し高い声 | 「わあ、見つけたね」 |
| ゆっくり | 「こ・れ・は、ボール」 |
| 抑揚が大きい | 「おっきいねえ」 |
| 短い文 | 「ワンワンいたね」 |
| 反応を待つ | 声や表情が返ってくるのを待つ |

これは、意味のない言葉を使うことではありません。  
赤ちゃんが聞き取りやすく、参加しやすい形にすることです。

### 毎日の語りかけテンプレート

```text
1. 赤ちゃんが見たものを見る
2. 名前を言う
3. 状態・動き・色・音を足す
4. 少し待つ
5. 赤ちゃんの声や表情に返す
```

| 赤ちゃんの行動 | 親の返し方 |
|---|---|
| 犬を見る | 「犬だね。白い犬だね。歩いてるね」 |
| コップを触る | 「コップだね。冷たいね」 |
| ボールを落とす | 「落ちたね。もう一回？」 |
| 指差す | 「あれが気になるんだね。電気だね」 |
| 喃語を出す | 「あーって言ったね。あー、楽しいね」 |

### やりすぎない

親が一日中話し続ける必要はありません。  
むしろ、赤ちゃんの反応を待つ間がないと、会話ターンになりません。

## 13-7. 読み聞かせ：0か月から始めてよい

### なぜ絵本が効くのか

0歳の読み聞かせは、内容を理解させるためだけではありません。  
次の効果が重なります。

| 要素 | 発達への意味 |
|---|---|
| 親の声 | 安心、音声への注意 |
| 同じものを見る | 共同注意 |
| ページをめくる | 順番、予測 |
| 繰り返し | 記憶、語彙 |
| 絵を指差す | 物の名前、視線共有 |
| 親子で座る | 愛着、落ち着き |

AAPは、出生直後から親子で読み聞かせを始めることを推奨しています<sup>[8](#ref-aap-reading)</sup>。家庭での共有読書は、語彙や表出語彙などの発達アウトカムと正の関連を示すメタ分析もあります<sup>[9](#ref-frontiers-shared-reading-2025)</sup>。

### 限界

読書研究には観察研究も多く含まれます。  
本を読む家庭は、そもそも親子の会話、生活リズム、教育資源が豊かなこともあります。したがって「絵本だけで発達が決まる」とは読みません。

それでも、読み聞かせは低コストで、親子のやりとりを増やしやすく、副作用が少ない実践です。

### 年齢別の読み方

| 年齢 | 読み方 |
|---|---|
| 0〜3か月 | 親の声を聞かせる。短くてよい |
| 3〜6か月 | 絵を見せ、音やリズムを楽しむ |
| 6〜9か月 | 厚紙絵本を触らせる。なめてもよい本にする |
| 9〜12か月 | 指差しに名前を返す |
| 12〜18か月 | 「どこかな？」と探す |
| 18〜24か月 | 子どもの言葉を広げる。「ワンワン」→「大きいワンワンだね」 |

## 13-8. 遊びとおもちゃ：高価な知育玩具より親子の関わり

### 良いおもちゃの条件

AAPの玩具選択に関する臨床報告では、良いおもちゃは、養育者との遊び、言語、ふり遊び、問題解決、創造性、身体活動を促すものとされています<sup>[10](#ref-aap-toys)</sup>。

| 良いおもちゃ | 理由 |
|---|---|
| 積み木 | 積む、崩す、比べる、見立てる |
| コップ重ね | 大小、入れる、出す、順番 |
| ボール | 追視、転がす、投げる、取りに行く |
| 布、箱、容器 | 隠す、包む、入れる、出す |
| ままごと | 模倣、言葉、社会性 |
| 厚紙絵本 | 共同注意、語彙、指差し |

電子音や画面が多い玩具が、単純なおもちゃより発達に優れるとは限りません。  
むしろ、子どもが能動的に操作でき、親が言葉を添えやすいものが実用的です。

### おもちゃは多すぎない方がよい

36人の幼児を対象にした研究では、おもちゃが16個ある環境より4個の環境の方が、1つのおもちゃで遊ぶ時間が長く、遊び方も多様になりました<sup>[11](#ref-toy-number)</sup>。サンプルは大きくありませんが、家庭での実践としては「大量に出しっぱなしにしない」「少数をローテーションする」方針に合理性があります。

### 遊びの型

| 遊び | 育つ力 |
|---|---|
| いないいないばあ | 予測、記憶、社会的期待 |
| コップ重ね | 大小、空間、手指 |
| ボール転がし | 追視、原因結果、移動意欲 |
| 積み木 | 空間認知、問題解決、創造性 |
| 型はめ | 試行錯誤、形の認識 |
| ままごと | 模倣、言語、社会性 |
| 絵本の指差し | 共同注意、語彙 |
| 歌遊び | リズム、予測、情動調整 |

0〜2歳の遊びは、正解を当てることではありません。  
何度も試す、大人と一緒に面白がる、失敗してもまたできる環境が重要です。

## 13-9. 運動発達：床で動く時間が知育の土台

### タミータイム

タミータイムは、赤ちゃんが起きていて、大人が見守っている時に、腹ばいで過ごす時間です。  
寝る時は仰向けですが、起きている時の腹ばいは、首、肩、背中、体幹の発達、頭の形の偏り予防に役立ちます。

2020年のシステマティックレビューでは、タミータイムは粗大運動発達、総合発達、寝返り、腹ばい移動、ハイハイ、斜頭・短頭の予防と正の関連を示しました<sup>[12](#ref-tummy)</sup>。一方で、認知、社会性、微細運動への効果は不確実な領域もあります。

家庭での位置づけは、「万能知育」ではなく、運動と姿勢の基盤作りです。

### ハイハイの意味

ハイハイは、歩行前の単なる通過点ではありません。  
赤ちゃんが自分で移動できるようになると、世界との関わり方が変わります。

| 視点 | ハイハイで育つこと |
|---|---|
| 身体 | 肩、腕、手首、体幹、股関節 |
| 感覚 | 手のひら、膝、足裏から床情報を得る |
| 空間認知 | 距離、段差、障害物 |
| 視覚 | 近くと遠く、奥行き |
| 認知 | 目的地までどう行くか考える |
| 社会性 | 親の顔を見て安全確認する |
| 自己効力感 | 自分で世界に近づける |

Adolphらの研究では、乳児の移動経験が段差や高さに対する判断と関連すること、ハイハイで学んだ判断が歩行開始後にそのまま完全移行するわけではなく、歩く身体になった後に再び学ぶ必要があることが示されています<sup>[13](#ref-adolph-cliff)</sup>。運動発達は月齢だけでなく、身体、環境、経験の相互作用で進みます。

### やってはいけない運動教育

| 避けること | 理由 |
|---|---|
| 早く立たせる練習を無理にする | 床での探索時間が減る |
| 長時間バウンサー・椅子に固定する | 自由運動が減る |
| 歩行器で歩行を促す | 事故リスクがあり、歩行を早める根拠は乏しい |
| ハイハイを飛ばさせようとする | 移動探索の経験が減る |
| できない動作を何度も強制する | 苦手意識、身体の過緊張につながる |

AAPは、乳児用歩行器について重大な事故リスクがあり、歩行を早めるどころか発達を遅らせる可能性もあるとして使用を勧めていません<sup>[14](#ref-aap-walkers)</sup>。

## 13-10. スクリーン・動画教材

### 乳幼児は画面から学びにくい

乳幼児は、画面上の情報よりも、目の前の人との相互作用から学びやすい傾向があります。  
これはvideo deficitと呼ばれます。0〜6歳を対象にしたメタ分析でも、子どもは映像より対面からの方が学びやすい傾向が示されています<sup>[15](#ref-video-deficit)</sup>。

WHOは、1歳未満のスクリーン時間を推奨せず、2歳でも座位スクリーン時間は少ないほどよいとしています<sup>[16](#ref-who-activity)</sup>。AAPも18か月未満はビデオ通話を除きメディア利用を避け、18〜24か月では高品質な内容を親と一緒に見ることを勧めています<sup>[17](#ref-aap-media)</sup>。

### 背景テレビにも注意

子どもが見ていなくても、部屋でテレビが流れていると、親子の会話や遊びの質が下がる可能性があります。背景テレビが親子相互作用に影響することを示した研究もあります<sup>[18](#ref-background-tv)</sup>。

### 家庭ルール

| 年齢 | ルール |
|---|---|
| 0〜18か月 | 原則なし。ビデオ通話は可 |
| 18〜24か月 | 使うなら短時間、高品質、親と一緒 |
| 2歳以降 | 1時間以内を目安。食事中・寝る前は避ける |
| 全年齢 | 背景テレビをつけっぱなしにしない |

動画教材は悪ではありません。  
ただし、0〜2歳では、動画を見せるより、親が同じものを見て話す方が教育効果を期待しやすいと考えます。

## 13-11. 歌・音楽・リズム遊び

### 受け身の音楽より、親子で歌う

クラシック音楽を流すだけで知能が上がる、という単純な話ではありません。  
研究上より有望なのは、親子で歌う、リズムを取る、手遊びをするなど、能動的な音楽参加です。

6か月児を対象にしたランダム化研究では、能動的な音楽クラスに参加した乳児は、受け身の音楽曝露群に比べ、音楽的理解だけでなく、前言語的コミュニケーションや社会的行動に良い変化が示されました<sup>[19](#ref-active-music)</sup>。母子の歌唱が親の感情や親子の情緒的近さに関わる可能性を示す研究もあります<sup>[20](#ref-mother-infant-singing)</sup>。

ただし、音楽は知能を上げる教材ではありません。  
親子の情緒、リズム、やりとりを増やす手段です。

### 月齢別の音楽遊び

| 月齢 | 音楽遊び |
|---|---|
| 0〜3か月 | 抱っこでゆっくり歌う |
| 3〜6か月 | 手足をやさしく動かしながら歌う |
| 6〜9か月 | いないいないばあ、手遊び歌 |
| 9〜12か月 | 拍手、バイバイ、リズムまね |
| 1〜2歳 | 踊る、止まる、まねる、楽器を鳴らす |

## 13-12. バイリンガル・英語教育

### 複数言語は混乱の原因ではない

家庭で複数言語を使うこと自体は、子どもを混乱させるものではありません。  
AAP系の保護者向け情報では、多言語環境は子どもを混乱させず、発達過程で言語が混在することもあると説明されています<sup>[21](#ref-aap-bilingual)</sup>。

### 重要なのは人とのやりとり

英語を早く始めることより、人とやりとりしながら使うことが重要です。  
録音音声や動画を流すだけでは、乳幼児の学習には限界があります。二言語環境を作るなら、親、祖父母、保育者など、実際の人が自然に話しかける形が望ましいです<sup>[22](#ref-cal-bilingual)</sup>。

| 家庭の状況 | 方針 |
|---|---|
| 親が英語を自然に話せる | 日常の一部として使う |
| 片親が別言語を話す | それぞれの言語で自然に関わる |
| 親が英語に自信がない | 絵本・歌を楽しむ程度で十分 |
| 動画教材中心にしたい | 0〜2歳では優先度を下げる |
| 日本語が遅れそうで不安 | まず親子の会話量と反応を増やし、必要なら相談 |

## 13-13. 親のメンタルヘルスと複数養育者

赤ちゃんの発達教育を考える時、親のメンタルヘルスは周辺要素ではありません。  
親が疲弊し、抑うつや不安が強い状態では、赤ちゃんのサインに気づく余裕や応答性が下がることがあります。母親の抑うつが養育者の応答性や乳児の表出言語と関連することを示す研究もあります<sup>[23](#ref-maternal-depression)</sup>。

WHOの早期発達ガイドラインでも、乳幼児の発達支援は、応答的養育、早期学習、栄養、養育者のメンタルヘルスを組み合わせて考えるものとされています<sup>[2](#ref-who-ecd-guideline)</sup>。

父親、パートナー、祖父母、保育者など、複数の大人が安定して関わることも発達環境を豊かにします。父親の関与と子どもの認知発達に関するレビューでは、父親の関わりが子どもの認知発達と関連する可能性が示されています<sup>[24](#ref-father-involvement)</sup>。

重要なのは、母親と同じやり方を完全に再現することではありません。  
複数の大人が、それぞれのスタイルで、温かく、予測可能に関わることです。

## 13-14. ポジティブなしつけ

0〜2歳では、まだ言葉で長く説明して理解する力は未熟です。  
だからこそ、叱るよりも、環境を整え、短い言葉で止め、代替行動を示します。

体罰に関するメタ分析では、スパンキングは望ましくない子どものアウトカムと関連し、長期的な肯定的効果は示されていません<sup>[25](#ref-spanking-meta)</sup>。AAPも、叩く、脅す、侮辱する、恥をかかせるようなしつけを避け、肯定的なしつけを推奨しています<sup>[26](#ref-aap-discipline)</sup>。

### 0〜2歳のしつけは「感情の代弁＋環境調整」

| 場面 | 対応 |
|---|---|
| 物を投げる | 「投げたいね。これは投げない。ボールを投げよう」 |
| 叩く | 手を止めて「叩かない。痛い。なでなでしよう」 |
| 触ってほしくない物に行く | 叱る前に届かない場所へ移す |
| 癇癪 | 安全確保、短い言葉、落ち着くまで待つ |
| 食べない | 無理に押し込まず、量と環境を調整する |

この時期の目標は、親の言うことを完全に聞かせることではありません。  
安全な範囲で欲求を受け止め、境界線を短く一貫して伝えることです。

## 13-15. 月齢別：発達を促す実践メニュー

### 0〜3か月

#### 目標

安心、睡眠、授乳、親の声、顔を見る経験を増やす。

#### 毎日やること

- [ ] 顔を見て話す
- [ ] 泣いたら抱く、声をかける
- [ ] 短い絵本を読む
- [ ] 抱っこで歌う
- [ ] 起きている時に短いタミータイム
- [ ] 左右から声をかけ、向き癖を減らす

#### やりすぎないこと

- [ ] 長時間の刺激
- [ ] 動画視聴
- [ ] 泣いても放置して鍛える考え方
- [ ] 早期教材の詰め込み

### 3〜6か月

#### 目標

見る、聞く、手を伸ばす、寝返り準備、声のやりとり。

#### 毎日やること

- [ ] 赤ちゃんの喃語をまねる
- [ ] 手を伸ばせる位置におもちゃを置く
- [ ] 床で自由に動く時間を作る
- [ ] 鏡やオーボールで腹ばい遊び
- [ ] 絵本を同じものでも何度も読む
- [ ] 親が実況中継する

#### ポイント

親がすぐに全部やってあげるより、赤ちゃんが少し手を伸ばす、少し体をひねる、少し待つ時間を作ると、運動と認知が同時に育ちます。

### 6〜9か月

#### 目標

ずりばい、座位、手指操作、離乳食、探索。

#### 毎日やること

- [ ] 床の安全地帯を作る
- [ ] 取りたい場所におもちゃを置く
- [ ] コップ重ね、布、ボールで遊ぶ
- [ ] いないいないばあ
- [ ] 離乳食中に「にんじんだね」「やわらかいね」と話す
- [ ] 背景テレビを消す

#### ポイント

この時期の教育は、机上より床です。  
赤ちゃんが自分で移動し、手を伸ばし、口で確かめ、安全に失敗できる環境を作ります。

### 9〜12か月

#### 目標

ハイハイ、つかまり立ち、指差し、模倣、共同注意。

#### 毎日やること

- [ ] 指差しに必ず反応する
- [ ] 「ちょうだい」「どうぞ」のやりとり
- [ ] 拍手、バイバイ、いただきます
- [ ] くるくる落とす、入れる、出す遊び
- [ ] 階段、段差、家具の安全対策
- [ ] ハイハイできるスペースを残す

#### ポイント

指差しは、単なる動作ではなく「一緒に見て」「名前を教えて」という強い学習サインです。  
指差しに親が返すことで、共同注意と言語学習が進みます。

### 12〜18か月

#### 目標

歩行、単語、まね、お手伝い、ごっこ遊びの始まり。

#### 毎日やること

- [ ] 子どもの単語を広げる
- [ ] 食事、着替え、片付けを言葉にする
- [ ] 積み木、型はめ、ままごとを入れる
- [ ] 外遊びで坂、砂、草、石、水を経験する
- [ ] 簡単なお手伝いを頼む
- [ ] 選択肢を2つにする

#### 例

```text
子「ワンワン」
親「大きいワンワンだね」
```

この時期は、生活全部が教材です。  
靴を履く、手を洗う、スプーンを使う、物を運ぶ、片付ける。そのすべてが運動、言語、自己効力感の学習になります。

### 18〜24か月

#### 目標

二語文、象徴遊び、イヤイヤ、自己主張、簡単なルール。

#### 毎日やること

- [ ] 気持ちを言葉にする
- [ ] ごっこ遊びを広げる
- [ ] 積み木、クレヨン、シール、粘土など手指遊びを入れる
- [ ] 外で走る、登る、くぐる
- [ ] 絵本を子どもに選ばせる
- [ ] できることは少し待つ

#### 例

```text
「嫌だったね」
「まだ遊びたかったね」
「先に靴を履く？帽子をかぶる？」
```

イヤイヤ期は教育失敗ではありません。  
自分の意思が育ってきた一方で、脳のブレーキがまだ未熟な時期です。親は「言い聞かせて止める」より、「安全を確保し、気持ちを代弁し、選択肢を出す」方が現実的です。

## 13-16. やらない方がよい発達教育

| やらない方がよいこと | 理由 |
|---|---|
| 0〜2歳に長時間動画を見せる | 対面相互作用、睡眠、身体活動を置き換える |
| 背景テレビをつけっぱなしにする | 親子の会話と遊びの質を下げる可能性 |
| 高額教材に任せる | 乳幼児は人とのやりとりから学びやすい |
| 早く座らせる、立たせる、歩かせる | 自由運動と探索が減る |
| 歩行器を使う | 事故リスクがあり、発達促進の根拠が乏しい |
| 玩具を出しすぎる | 集中した遊びが短くなる可能性 |
| 体罰、脅し、恥を使う | 長期的な有益性がなく、悪影響と関連 |
| 泣いても応答しない訓練をする | 0歳の安心と応答性の土台を損なう可能性 |
| 他児と比較して焦る | 発達には幅がある |
| フラッシュカード中心にする | 0〜2歳では対話、遊び、運動の優先度が高い |

## 13-17. 発達を伸ばす家庭の1日モデル

### 朝

```text
起床
↓
カーテンを開ける
↓
「朝だね。明るいね」と話す
↓
授乳・朝食
↓
床遊び
```

朝は、光、声、身体活動を入れます。  
乳児睡眠と認知・言語・実行機能には関連が示されていますが、多くは観察研究であり、睡眠だけで知能が決まるわけではありません<sup>[27](#ref-infant-sleep)</sup>。それでも、睡眠環境を整えることは、学習しやすい脳と身体を支える生活習慣です。

### 昼

```text
散歩
↓
外のものを実況
↓
帰宅して昼寝
↓
起きたら絵本
```

散歩は、視覚、聴覚、前庭感覚、言語入力の宝庫です。  
「車だね」「風が冷たいね」「鳥が鳴いたね」と、実体験に言葉を乗せます。

### 夕方

```text
床遊び
↓
手指遊び
↓
入浴
↓
保湿
↓
絵本
↓
就寝
```

寝る前は刺激を減らし、スクリーンを避け、同じ流れを繰り返します。  
0〜2歳の教育では、特別な教材時間よりも、毎日の生活の予測可能性が情緒の安定につながります。

## 13-18. 実践チェックリスト

### 毎日チェック

- [ ] 赤ちゃんの声・視線・指差しに返した
- [ ] 絵本を1冊以上読んだ
- [ ] 親が実況中継のように話しかけた
- [ ] 床で自由に動く時間を作った
- [ ] 起きている時に腹ばい・姿勢遊びをした
- [ ] 背景テレビを消した
- [ ] 抱っこ・歌・スキンシップを入れた
- [ ] 赤ちゃんが自分で試す時間を待った
- [ ] 親も休む時間を確保した

### 週1チェック

- [ ] おもちゃを出しすぎていない
- [ ] 絵本を手に取りやすい場所に置いた
- [ ] 床の安全地帯を更新した
- [ ] 外遊び・散歩の時間を取った
- [ ] 親の疲労が限界に近くないか確認した
- [ ] 発達で気になることをメモした

### 受診・相談の目安

発達には幅があります。  
ただし、心配な時は「様子を見る」だけで長く抱え込まず、小児科、乳幼児健診、自治体の発達相談につなぐ方が安心です。

CDCの発達マイルストーンは、子どもの遊び、学習、話す、行動、運動の節目を追跡し、心配があれば早期に相談するためのツールです<sup>[28](#ref-cdc-milestones)</sup>。日本では、自治体の乳幼児健診、小児科、保健師、発達相談の案内を優先します。

相談のために、次をメモします。

- [ ] 気になる行動
- [ ] いつから
- [ ] どの場面で
- [ ] できること
- [ ] できにくいこと
- [ ] 睡眠、食事、体調
- [ ] 写真や動画

## 13-19. 研究サーベイとしての結論

0〜2歳の発達教育で、最も再現性が高く、家庭で実践しやすいのは次のことです。

1. 赤ちゃんの反応に返す
2. 親子の会話ターンを増やす
3. 毎日読み聞かせる
4. 床で自由に動ける時間を作る
5. 遊びを通して探索させる
6. スクリーンを親子のやりとりの代替にしない
7. 親のメンタルヘルスと休息を守る
8. 叩く、怒鳴る、脅すしつけを避ける
9. 安全な環境を作り、危険を叱る前に取り除く
10. 高額教材より、毎日の生活を学習機会にする

一言でまとめるなら、0〜2歳の最高の教育は、親が先生になることではありません。  
赤ちゃんが安心して探索できる世界を作り、赤ちゃんの発見に一緒に驚くことです。
